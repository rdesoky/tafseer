إِنَّ الَّذِينَ قَالُوا رَبُّنَا اللَّهُ ثُمَّ اسْتَقَامُوا فَلَا خَوْفٌ عَلَيْهِمْ وَلَا هُمْ يَحْزَنُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّ الَّذِينَ قَالُوا رَبّنَا اللَّه ثُمَّ اِسْتَقَامُوا فَلَا خَوْف عَلَيْهِمْ وَلَا هُمْ يَحْزَنُونَ } يَقُول تَعَالَى ذِكْره : { إِنَّ الَّذِينَ قَالُوا رَبّنَا اللَّه } الَّذِي لَا إِلَه غَيْره { ثُمَّ اِسْتَقَامُوا } عَلَى تَصْدِيقهمْ بِذَلِكَ فَلَمْ يَخْلِطُوهُ بِشِرْكٍ , وَلَمْ يُخَالِفُوا اللَّه فِي أَمْره وَنَهْيه { فَلَا خَوْف عَلَيْهِمْ } مِنْ فَزَع يَوْم الْقِيَامَة وَأَهْوَاله { وَلَا هُمْ يَحْزَنُونَ } عَلَى مَا خَلَّفُوا وَرَاءَهُمْ بَعْد مَمَاتهمْ .
