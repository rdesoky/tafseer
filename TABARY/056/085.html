وَنَحْنُ أَقْرَبُ إِلَيْهِ مِنْكُمْ وَلَكِنْ لَا تُبْصِرُونَ

يَقُول : { وَنَحْنُ أَقْرَب إِلَيْهِ مِنْكُمْ } يَقُول : وَرُسُلنَا الَّذِينَ يَقْبِضُونَ رُوحه أَقْرَب إِلَيْهِ مِنْكُمْ , { وَلَكِنْ لَا تُبْصِرُونَ } . وَكَانَ بَعْض أَهْل الْعَرَبِيَّة مِنْ أَهْل الْبَصْرَة يَقُول : قِيلَ { فَلَوْلَا إِذَا بَلَغَتْ الْحُلْقُوم وَأَنْتُمْ حِينَئِذٍ تَنْظُرُونَ } كَأَنَّهُ قَدْ سَمِعَ مِنْهُمْ , وَاَللَّه أَعْلَم : إِنَّا نَقْدِر عَلَى أَنْ لَا نَمُوت , فَقَالَ : { فَلَوْلَا إِذَا بَلَغَتْ الْحُلْقُوم } , ثُمَّ قَالَ { فَلَوْلَا إِنْ كُنْتُمْ غَيْر مَدِينِينَ } أَيْ غَيْر مَجْزِيِّينَ تَرْجِعُونَ تِلْكَ النُّفُوس وَأَنْتُمْ تَرَوْنَ كَيْف تَخْرُج عِنْد ذَلِكَ إِنْ كُنْتُمْ صَادِقِينَ بِأَنَّكُمْ تَمْتَنِعُونَ مِنْ الْمَوْت .
