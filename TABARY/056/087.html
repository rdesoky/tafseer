تَرْجِعُونَهَا إِنْ كُنْتُمْ صَادِقِينَ

وَقَوْله : { تَرْجِعُونَهَا إِنْ كُنْتُمْ صَادِقِينَ } يَقُول : تَرُدُّونَ تِلْكَ النُّفُوس مِنْ بَعْد مَصِيرهَا إِلَى الْحَلَاقِيم إِلَى مُسْتَقَرّهَا مِنْ الْأَجْسَاد إِنْ كُنْتُمْ صَادِقِينَ , إِنْ كُنْتُمْ تَمْتَنِعُونَ مِنْ الْمَوْت وَالْحِسَاب وَالْمُجَازَاة , وَجَوَاب قَوْله : { فَلَوْلَا إِذَا بَلَغَتْ الْحُلْقُوم } , وَجَوَاب قَوْله : { فَلَوْلَا إِنْ كُنْتُمْ غَيْر مَدِينِينَ } جَوَاب وَاحِد وَهُوَ قَوْله : { تَرْجِعُونَهَا } وَذَلِكَ نَحْو قَوْله : { فَإِمَّا يَأْتِيَنَّكُمْ مِنِّي هُدًى فَمَنْ تَبِعَ هُدَايَ فَلَا خَوْف عَلَيْهِمْ } 2 38 جَعَلَ جَوَاب الْجَزَاءَيْنِ جَوَابًا وَاحِدًا . وَبِنَحْوِ الَّذِي قُلْنَا فِي قَوْله : { تَرْجِعُونَهَا } قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 25991 - حَدَّثَنِي يُونُس , قَالَ : أَخْبَرَنَا اِبْن وَهْب , قَالَ قَالَ اِبْن زَيْد , فِي قَوْله : { تَرْجِعُونَهَا } قَالَ : لِتِلْك النَّفْس { إِنْ كُنْتُمْ صَادِقِينَ } .
