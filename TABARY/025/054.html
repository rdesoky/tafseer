وَهُوَ الَّذِي خَلَقَ مِنَ الْمَاءِ بَشَرًا فَجَعَلَهُ نَسَبًا وَصِهْرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَهُوَ الَّذِي خَلَقَ مِنَ الْمَاء بَشَرًا فَجَعَلَهُ نَسَبًا وَصِهْرًا } . يَقُول تَعَالَى ذِكْره : وَاللَّه الَّذِي خَلَقَ مِنْ النُّطَف بَشَرًا إِنْسَانًا فَجَعَلَهُ نَسَبًا , وَذَلِكَ سَبْعَة , وَصِهْرًا , وَهُوَ خَمْسَة . كَمَا : 20063 - حُدِّثْت عَنِ الْحُسَيْن , قَالَ : سَمِعْت أَبَا مُعَاذ يَقُول : أَخْبَرَنَا عُبَيْد , قَالَ : سَمِعْت الضَّحَّاك يَقُول فِي قَوْله : { فَجَعَلَهُ نَسَبًا وَصِهْرًا } النَّسَب : سَبْع , قَوْله : { حُرِّمَتْ عَلَيْكُمْ أُمَّهَاتكُمْ } ... 4 23 إِلَى قَوْله { وَبَنَات الْأُخْت } 4 23 وَالصِّهْر خَمْس , قَوْله : { وَأُمَّهَاتكُمْ اللَّاتِي أَرْضَعْنَكُمْ } ... 4 23 إِلَى قَوْله { وَحَلَائِل أَبْنَائِكُمُ الَّذِينَ مِنْ أَصْلَابكُمْ } . 4 23

وَكَانَ رَبُّكَ قَدِيرًا

وَقَوْله : { وَكَانَ رَبّك قَدِيرًا } يَقُول : وَرَبّك يَا مُحَمَّد ذُو قُدْرَة عَلَى خَلْق مَا يَشَاء مِنْ الْخَلْق , وَتَصْرِيفهمْ فِيمَا شَاءَ وَأَرَادَ .
