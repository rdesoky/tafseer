أَرَأَيْتَ مَنِ اتَّخَذَ إِلَهَهُ هَوَاهُ أَفَأَنْتَ تَكُونُ عَلَيْهِ وَكِيلًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { أَرَأَيْت مَنِ اتَّخَذَ إِلَهه هَوَاهُ أَفَأَنْت تَكُون عَلَيْهِ وَكِيلًا } يَعْنِي تَعَالَى ذِكْره : { أَرَأَيْت } يَا مُحَمَّد { مَنِ اتَّخَذَ إِلَهه } شَهْوَته الَّتِي يَهْوَاهَا ; وَذَلِكَ أَنَّ الرَّجُل مِنَ الْمُشْرِكِينَ كَانَ يَعْبُد الْحَجَر , فَإِذَا رَأَى أَحْسَن مِنْهُ رَمَى بِهِ , وَأَخَذَ الْآخَر يَعْبُدهُ , فَكَانَ مَعْبُوده وَإِلَهه مَا يَتَخَيَّرهُ لِنَفْسِهِ ; فَلِذَلِكَ قَالَ جَلَّ ثَنَاؤُهُ { أَرَأَيْت مَنِ اتَّخَذَ إِلَهه هَوَاهُ أَفَأَنْت تَكُون عَلَيْهِ وَكِيلًا } يَقُول تَعَالَى ذِكْره : أَفَأَنْت تَكُون يَا مُحَمَّد عَلَى هَذَا حَفِيظًا فِي أَفْعَاله مَعَ عَظِيم جَهْله ؟
