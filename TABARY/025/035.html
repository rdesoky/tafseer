وَلَقَدْ آتَيْنَا مُوسَى الْكِتَابَ وَجَعَلْنَا مَعَهُ أَخَاهُ هَارُونَ وَزِيرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَقَدْ آتَيْنَا مُوسَى الْكِتَاب وَجَعَلْنَا مَعَهُ أَخَاهُ هَارُون وَزِيرًا } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ يَتَوَعَّد مُشْرِكِي قَوْمه عَلَى كُفْرهمْ بِاللَّهِ , وَتَكْذِيبهمْ رَسُوله وَيُخَوِّفهُمْ مِنْ حُلُول نِقْمَته بِهِمْ , نَظِير الَّذِي يَحِلّ بِمَنْ كَانَ قَبْلهمْ مِنْ الْأُمَم الْمُكَذِّبَة رُسُلهَا : { وَلَقَدْ آتَيْنَا } يَا مُحَمَّد { مُوسَى الْكِتَاب } يَعْنِي التَّوْرَاة , كَالَّذِي آتَيْنَاك مِنَ الْفُرْقَان { وَجَعَلْنَا مَعَهُ أَخَاهُ هَارُون وَزِيرًا } يَعْنِي : مُعِينًا وَظَهِيرًا
