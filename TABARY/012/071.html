قَالُوا وَأَقْبَلُوا عَلَيْهِمْ مَاذَا تَفْقِدُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالُوا وَأَقْبَلُوا عَلَيْهِمْ مَاذَا تَفْقِدُونَ } يَقُول تَعَالَى ذِكْره : قَالَ بَنُو يَعْقُوب لَمَّا نُودُوا : { أَيَّتهَا الْعِير إِنَّكُمْ لَسَارِقُونَ } وَأَقْبَلُوا عَلَى الْمُنَادِي وَمَنْ بِحَضْرَتِهِمْ يَقُولُونَ لَهُمْ : { مَاذَا تَفْقِدُونَ } مَا الَّذِي تَفْقِدُونَ ؟ { قَالُوا نَفْقِد صُوَاع الْمَلِك } يَقُول : فَقَالَ لَهُمْ الْقَوْم : نَفْقِد مَشْرَبَة الْمَلِك .
