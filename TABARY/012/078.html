قَالُوا يَا أَيُّهَا الْعَزِيزُ إِنَّ لَهُ أَبًا شَيْخًا كَبِيرًا فَخُذْ أَحَدَنَا مَكَانَهُ إِنَّا نَرَاكَ مِنَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى . { قَالُوا يَا أَيّهَا الْعَزِيز إِنَّ لَهُ أَبًا شَيْخًا كَبِيرًا فَخُذْ أَحَدَنَا مَكَانَهُ إِنَّا نَرَاك مِنَ الْمُحْسِنِينَ } يَقُول تَعَالَى ذِكْره : قَالَتْ إِخْوَة يُوسُف لِيُوسُف : { يَا أَيُّهَا الْعَزِيز } يَا أَيّهَا الْمَلِك { إِنَّ لَهُ أَبًا شَيْخًا كَبِيرًا } كَلِفًا بِحُبِّهِ , يَعْنُونَ يَعْقُوب . { فَخُذْ أَحَدَنَا مَكَانَهُ } يَعْنُونَ فَخُذْ أَحَدًا مِنَّا بَدَلًا مِنْ بِنْيَامِين , وَخَلِّ عَنْهُ . { إِنَّا نَرَاك مِنَ الْمُحْسِنِينَ } يَقُولُونَ : إِنَّا نَرَاك مِنَ الْمُحْسِنِينَ فِي أَفْعَالِك . وَقَالَ مُحَمَّد بْن إِسْحَاق فِي ذَلِكَ , مَا . 14984 - حَدَّثَنَا ابْن حُمَيْد , قَالَ : ثنا سَلَمَة , عَنِ ابْن إِسْحَاق : { إِنَّا نَرَاك مِنَ الْمُحْسِنِينَ } إِنَّا نَرَى ذَلِكَ مِنْك إِحْسَانًا إِنْ فَعَلْت
