وَلَوْ شِئْنَا لَآتَيْنَا كُلَّ نَفْسٍ هُدَاهَا وَلَكِنْ حَقَّ الْقَوْلُ مِنِّي لَأَمْلَأَنَّ جَهَنَّمَ مِنَ الْجِنَّةِ وَالنَّاسِ أَجْمَعِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَوْ شِئْنَا لَآتَيْنَا كُلَّ نَفْس هُدَاهَا وَلَكِنْ حَقَّ الْقَوْل مِنِّي لَأَمْلَأَنَّ جَهَنَّمَ مِنَ الْجِنَّة وَالنَّاس أَجْمَعِينَ } يَقُول تَعَالَى ذِكْره : { وَلَوْ شِئْنَا } يَا مُحَمَّد { لَآتَيْنَا } هَؤُلَاءِ الْمُشْرِكِينَ بِاللَّهِ مِنْ قَوْمك وَغَيْرهمْ مِنْ أَهْل الْكُفْر بِاللَّهِ { هُدَاهَا } يَعْنِي : رُشْدهَا وَتَوْفِيقهَا لِلْإِيمَانِ بِاللَّهِ { وَلَكِنْ حَقَّ الْقَوْل مِنِّي } يَقُول : وَجَبَ الْعَذَاب مِنِّي لَهُمْ , وَقَوْله { لَأَمْلَأَنَّ جَهَنَّم مِنَ الْجِنَّة وَالنَّاس أَجْمَعِينَ } يَعْنِي مِنْ أَهْل الْمَعَاصِي وَالْكُفْر بِاللَّهِ مِنْهُمْ . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 21502 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { وَلَوْ شِئْنَا لَآتَيْنَا كُلَّ نَفْس هُدَاهَا } قَالَ : لَوْ شَاءَ اللَّه لَهَدَى النَّاسَ جَمِيعًا , لَوْ شَاءَ اللَّه لَأَنْزَلَ { عَلَيْهِمْ مِنَ السَّمَاء آيَة فَظَلَّتْ أَعْنَاقهمْ لَهَا خَاضِعِينَ } 26 4 { وَلَكِنْ حَقّ الْقَوْل مِنِّي } حَقّ الْقَوْل عَلَيْهِمْ .
