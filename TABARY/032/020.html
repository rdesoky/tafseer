وَأَمَّا الَّذِينَ فَسَقُوا

وَقَوْله : { وَأَمَّا الَّذِينَ فَسَقُوا } يَقُول تَعَالَى ذِكْره : وَأَمَّا الَّذِينَ كَفَرُوا بِاللَّهِ , وَفَارَقُوا طَاعَتَهُ

فَمَأْوَاهُمُ النَّارُ

{ فَمَأْوَاهُمُ النَّار } يَقُول : فَمَسَاكِنهمْ الَّتِي يَأْوُونَ إِلَيْهَا فِي الْآخِرَة النَّار

كُلَّمَا أَرَادُوا أَنْ يَخْرُجُوا مِنْهَا أُعِيدُوا فِيهَا وَقِيلَ لَهُمْ ذُوقُوا عَذَابَ النَّارِ الَّذِي كُنْتُمْ بِهِ تُكَذِّبُونَ

{ كُلَّمَا أَرَادُوا أَنْ يَخْرُجُوا مِنْهَا أُعِيدُوا فِيهَا وَقِيلَ لَهُمْ ذُوقُوا عَذَاب النَّار الَّذِي كُنْتُمْ بِهِ } فِي الدُّنْيَا { تُكَذِّبُونَ } أَنَّ اللَّه أَعَدَّهَا لِأَهْلِ الشِّرْك بِهِ . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 21543 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { وَأَمَّا الَّذِينَ فَسَقُوا } أَشْرَكُوا { وَقِيلَ لَهُمْ ذُوقُوا عَذَاب النَّار الَّذِي كُنْتُمْ بِهِ تُكَذِّبُونَ } وَالْقَوْم مُكَذِّبُونَ كَمَا تَرَوْنَ .
