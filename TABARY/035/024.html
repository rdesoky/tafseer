إِنَّا أَرْسَلْنَاكَ بِالْحَقِّ بَشِيرًا وَنَذِيرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّا أَرْسَلْنَاك بِالْحَقِّ بَشِيرًا وَنَذِيرًا } يَقُول جَلَّ ثَنَاؤُهُ لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { إِنَّا أَرْسَلْنَاك } يَا مُحَمَّد { بِالْحَقِّ } وَهُوَ الْإِيمَان بِاللَّهِ وَشَرَائِع الدِّين الَّتِي افْتَرَضَهَا عَلَى عِبَاده { بَشِيرًا } يَقُول : مُبَشِّرًا بِالْجَنَّةِ مَنْ صَدَّقَك وَقَبِلَ مِنْك مَا جِئْت بِهِ مِنْ عِنْد اللَّه مِنَ النَّصِيحَة { وَنَذِيرًا } تُنْذِر النَّاسَ مَنْ كَذَّبَك وَرَدَّ عَلَيْك مَا جِئْت بِهِ مِنْ عِنْد اللَّه مِنْ النَّصِيحَة .

وَإِنْ مِنْ أُمَّةٍ إِلَّا خَلَا فِيهَا نَذِيرٌ

{ وَإِنْ مِنْ أُمَّة إِلَّا خَلَا فِيهَا نَذِير } يَقُول : وَمَا مِنْ أُمَّة مِنَ الْأُمَم الدَّائِنَة بِمِلَّةٍ إِلَّا خَلَا فِيهَا مِنْ قَبْلِك نَذِير يُنْذِرهُمْ بَأْسَنَا عَلَى كُفْرهمْ بِاللَّهِ , كَمَا : 22163 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { وَإِنْ مِنْ أُمَّة إِلَّا خَلَا فِيهَا نَذِير } كُلّ أُمَّة كَانَ لَهَا رَسُول
