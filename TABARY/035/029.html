إِنَّ الَّذِينَ يَتْلُونَ كِتَابَ اللَّهِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّ الَّذِينَ يَتْلُونَ كِتَابَ اللَّه } يَقُول تَعَالَى ذِكْره : إِنَّ الَّذِينَ يَقْرَءُونَ كِتَابَ اللَّه الَّذِي أَنْزَلَهُ عَلَى مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ { وَأَقَامُوا الصَّلَاةَ } يَقُول : وَأَدَّوْا الصَّلَاةَ الْمَفْرُوضَةَ لِمَوَاقِيتِهَا بِحُدُودِهَا .

وَأَقَامُوا الصَّلَاةَ

وَقَالَ : وَأَقَامُوا الصَّلَاةَ بِمَعْنَى : وَيُقِيمُوا الصَّلَاة .

وَأَنْفَقُوا مِمَّا رَزَقْنَاهُمْ سِرًّا وَعَلَانِيَةً

وَقَوْله : { وَأَنْفَقُوا مِمَّا رَزَقْنَاهُمْ سِرًّا وَعَلَانِيَةً } يَقُول : وَتَصَدَّقُوا بِمَا أَعْطَيْنَاهُمْ مِنْ الْأَمْوَال سِرًّا فِي خَفَاء , وَعَلَانِيَةً : جِهَارًا , وَإِنَّمَا مَعْنَى ذَلِكَ أَنَّهُمْ يُؤَدُّونَ الزَّكَاةَ الْمَفْرُوضَةَ , وَيَتَطَوَّعُونَ أَيْضًا بِالصَّدَقَةِ مِنْهُ بَعْد أَدَاء الْفَرْض الْوَاجِب عَلَيْهِمْ فِيهِ.

يَرْجُونَ تِجَارَةً لَنْ تَبُورَ

وَقَوْله : { يَرْجُونَ تِجَارَةً لَنْ تَبُور } يَقُول تَعَالَى ذِكْره : يَرْجُونَ بِفِعْلِهِمْ ذَلِكَ تِجَارَةً لَنْ تَبُورَ : لَنْ تَكْسُد وَلَنْ تَهْلِك ; مِنْ قَوْلهمْ : بَارَتْ السُّوق : إِذَا كَسَدَتْ وَبَارَ الطَّعَام , وَقَوْله : { تِجَارَة } جَوَاب لِأَوَّلِ الْكَلَام .
