وَالَّذِي أَوْحَيْنَا إِلَيْكَ مِنَ الْكِتَابِ هُوَ الْحَقُّ مُصَدِّقًا لِمَا بَيْنَ يَدَيْهِ إِنَّ اللَّهَ بِعِبَادِهِ لَخَبِيرٌ بَصِيرٌ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَالَّذِي أَوْحَيْنَا إِلَيْك مِنَ الْكِتَاب هُوَ الْحَقّ مُصَدِّقًا لِمَا بَيْنَ يَدَيْهِ إِنَّ اللَّهَ بِعِبَادِهِ لَخَبِير بَصِير } يَقُول تَعَالَى ذِكْره : { وَالَّذِي أَوْحَيْنَا إِلَيْك مِنَ الْكِتَاب } يَا مُحَمَّد , وَهُوَ هَذَا الْقُرْآن الَّذِي أَنْزَلَهُ اللَّه عَلَيْهِ { هُوَ الْحَقّ } يَقُول : هُوَ الْحَقّ عَلَيْك وَعَلَى أُمَّتك أَنْ تَعْمَل بِهِ , وَتَتَّبِع مَا فِيهِ دُونَ غَيْره مِنَ الْكُتُب الَّتِي أَوْحَيْت إِلَى غَيْرك { مُصَدِّقًا لِمَا بَيْن يَدَيْهِ } يَقُول : هُوَ يُصَدِّق مَا مَضَى بَيْنَ يَدَيْهِ , فَصَارَ أَمَامه مِنْ الْكُتُب الَّتِي أَنْزَلْتهَا إِلَى مَنْ قَبْلَك مِنْ الرُّسُل , كَمَا : 22172 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة , قَوْله : { وَالَّذِي أَوْحَيْنَا إِلَيْك مِنَ الْكِتَاب هُوَ الْحَقّ مُصَدِّقًا لِمَا بَيْنَ يَدَيْهِ } لِلْكُتُبِ الَّتِي خَلَتْ قَبْلَهُ وَقَوْله : { إِنَّ اللَّهَ بِعِبَادِهِ لَخَبِير بَصِير } يَقُول تَعَالَى ذِكْره : إِنَّ اللَّهَ بِعِبَادِهِ لَذُو عِلْم وَخِبْرَة بِمَا يَعْمَلُونَ بَصِير بِمَا يُصْلِحهُمْ مِنْ التَّدْبِير .
