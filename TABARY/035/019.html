وَمَا يَسْتَوِي الْأَعْمَى وَالْبَصِيرُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَمَا يَسْتَوِي الْأَعْمَى وَالْبَصِير } يَقُول تَعَالَى ذِكْره : { وَمَا يَسْتَوِي الْأَعْمَى } عَنْ دِين اللَّه الَّذِي ابْتَعَثَ بِهِ نَبِيّه مُحَمَّدًا صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ { وَالْبَصِير } الَّذِي قَدْ أَبْصَرَ فِيهِ رَشَدَهُ , فَاتَّبَعَ مُحَمَّدًا وَصَدَّقَهُ , وَقَبِلَ عَنِ اللَّه مَا ابْتَعَثَهُ بِهِ
