مَلْعُونِينَ أَيْنَمَا ثُقِفُوا أُخِذُوا وَقُتِّلُوا تَقْتِيلًا

وَقَوْله : { مَلْعُونِينَ أَيْنَمَا ثُقِفُوا أُخِذُوا وَقُتِّلُوا تَقْتِيلًا } يَقُول تَعَالَى ذِكْره : مَطْرُودِينَ مَنْفِيِّينَ { أَيْنَمَا ثُقِفُوا } يَقُول : حَيْثُمَا لُقُوا مِنَ الْأَرْض أُخِذُوا وَقُتِّلُوا لِكُفْرِهِمْ بِاللَّهِ تَقْتِيلًا . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 21877 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { مَلْعُونِينَ } عَلَى كُلّ حَال { أَيْنَمَا ثُقِفُوا } أُخِذُوا { وَقُتِّلُوا تَقْتِيلًا } إِذَا هُمْ أَظْهَرُوا النِّفَاقَ . وَنُصِبَ قَوْله : { مَلْعُونِينَ } عَلَى الشَّتْم , وَقَدْ يَجُوز أَنْ يَكُونَ الْقَلِيل مِنْ صِفَة الْمَلْعُونِينَ , فَيَكُون قَوْله مَلْعُونِينَ مَرْدُودًا عَلَى الْقَلِيل , فَيَكُون مَعْنَاهُ : ثُمَّ لَا يُجَاوِرُونَك فِيهَا إِلَّا أَقِلَّاء مَلْعُونِينَ يُقَتَّلُونَ حَيْثُ أُصِيبُوا .
