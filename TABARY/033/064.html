إِنَّ اللَّهَ لَعَنَ الْكَافِرِينَ وَأَعَدَّ لَهُمْ سَعِيرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّ اللَّهَ لَعَنَ الْكَافِرِينَ وَأَعَدَّ لَهُمْ سَعِيرًا } يَقُول تَعَالَى ذِكْره : إِنَّ اللَّهَ أَبْعَدَ الْكَافِرِينَ بِهِ مِنْ كُلّ خَيْر , وَأَقْصَاهُمْ عَنْهُ { وَأَعَدَّ لَهُمْ سَعِيرًا } يَقُول : وَأَعَدَّ لَهُمْ فِي الْآخِرَة نَارًا تَتَّقِد وَتَتَسَعَّر لِيُصْلِيَهُمُوهَا { خَالِدِينَ فِيهَا أَبَدًا } يَقُول : مَاكِثِينَ فِي السَّعِير أَبَدًا , إِلَى غَيْر نِهَايَة { لَا يَجِدُونَ وَلِيًّا } يَتَوَلَّاهُمْ , فَيَسْتَنْقِذهُمْ مِنَ السَّعِير الَّتِي أَصْلَاهُمُوهَا اللَّه { وَلَا نَصِيرًا } يَنْصُرهُمْ , فَيُنْجِيهِمْ مِنْ عِقَاب اللَّه إِيَّاهُمْ .
