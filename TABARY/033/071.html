يُصْلِحْ لَكُمْ أَعْمَالَكُمْ وَيَغْفِرْ لَكُمْ ذُنُوبَكُمْ وَمَنْ يُطِعِ اللَّهَ وَرَسُولَهُ فَقَدْ فَازَ فَوْزًا عَظِيمًا

وَقَوْله : { يُصْلِح لَكُمْ أَعْمَالَكُمْ } يَقُول تَعَالَى ذِكْره لِلْمُؤْمِنِينَ : اتَّقُوا اللَّهَ وَقُولُوا السَّدَاد مِنْ الْقَوْل يُوَفِّقكُمْ لِصَالِحِ الْأَعْمَال , فَيُصْلِح أَعْمَالَكُمْ { وَيَغْفِر لَكُمْ ذُنُوبَكُمْ } يَقُول : وَيَعْفُ لَكُمْ عَنْ ذُنُوبكُمْ , فَلَا يُعَاقِبكُمْ عَلَيْهَا { وَمَنْ يُطِعِ اللَّهَ وَرَسُولَهُ } فَيَعْمَل بِمَا أَمَرَهُ بِهِ , وَيَنْتَهِي عَمَّا نَهَاهُ , وَيَقُلْ السَّدِيدَ { فَقَدْ فَازَ فَوْزًا عَظِيمًا } يَقُول : فَقَدْ ظَفِرَ بِالْكَرَامَةِ الْعُظْمَى مِنَ اللَّه .
