كَبُرَ مَقْتًا عِنْدَ اللَّهِ أَنْ تَقُولُوا مَا لَا تَفْعَلُونَ

وَاخْتَلَفَتْ أَهْل الْعَرَبِيَّة فِي مَعْنَى ذَلِكَ , وَفِي وَجْه نَصْب قَوْله : { كَبُرَ مَقْتًا } فَقَالَ بَعْض نَحْوِيِّي الْبَصْرَة : قَالَ : { كَبُرَ مَقْتًا عِنْد اللَّه } : أَيْ كَبُرَ مَقْتكُمْ مَقْتًا , ثُمَّ قَالَ : { أَنْ تَقُولُوا مَا لَا تَفْعَلُونَ } : أَذَى قَوْلكُمْ. وَقَالَ بَعْض نَحْوِيِّي الْكُوفَة : قَوْله : { يَا أَيّهَا الَّذِينَ آمَنُوا لِمَ تَقُولُونَ مَا لَا تَفْعَلُونَ } : كَانَ الْمُسْلِمُونَ يَقُولُونَ : لَوْ نَعْلَم أَيّ الْأَعْمَال أَحَبّ إِلَى اللَّه لَأَتَيْنَاهُ , وَلَوْ ذَهَبَتْ فِيهِ أَنْفُسنَا وَأَمْوَالنَا ; فَلَمَّا كَانَ يَوْم أُحُد , نَزَلُوا عَنْ النَّبِيّ صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ حَتَّى شُجَّ , وَكُسِرَتْ رَبَاعِيَته , فَقَالَ : { لِمَ تَقُولُونَ مَا لَا تَفْعَلُونَ } , ثُمَّ قَالَ : { كَبُرَ مَقْتًا عِنْد اللَّه } كَبُرَ ذَلِكَ مَقْتًا : أَيْ فَأَنْ فِي مَوْضِع رَفْع , لِأَنَّ كَبُرَ كَقَوْلِهِ : بِئْسَ رَجُلًا أَخُوك . وَقَوْله : { كَبُرَ مَقْتًا عِنْد اللَّه } وَعِنْد الَّذِينَ آمَنُوا , أُضْمِرَ فِي كَبُرَ اِسْم يَكُون مَرْفُوعًا . وَالصَّوَاب مِنْ الْقَوْل فِي ذَلِكَ عِنْدِي أَنَّ قَوْله { مَقْتًا } مَنْصُوب عَلَى التَّفْسِير , كَقَوْلِ الْقَائِل : كَبُرَ قَوْلًا هَذَا الْقَوْل .
