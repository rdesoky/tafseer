وَإِذْ قَالَ مُوسَى لِقَوْمِهِ يَا قَوْمِ لِمَ تُؤْذُونَنِي وَقَدْ تَعْلَمُونَ أَنِّي رَسُولُ اللَّهِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَإِذْ قَالَ مُوسَى لِقَوْمِهِ يَا قَوْم لِمَ تُؤْذُونَنِي وَقَدْ تَعْلَمُونَ أَنِّي رَسُول اللَّه إِلَيْكُمْ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : وَاذْكُرْ يَا مُحَمَّد { إِذْ قَالَ مُوسَى } بْن عِمْرَان { لِقَوْمِهِ يَا قَوْم لِمَ تُؤْذُونَنِي وَقَدْ تَعْلَمُونَ } حَقًّا { أَنِّي رَسُول اللَّه إِلَيْكُمْ }

إِلَيْكُمْ فَلَمَّا زَاغُوا أَزَاغَ اللَّهُ

وَقَوْله : { فَلَمَّا زَاغُوا أَزَاغَ اللَّه قُلُوبهمْ } يَقُول : فَلَمَّا عَدَلُوا وَجَارُوا عَنْ قَصْد السَّبِيل أَزَاغَ اللَّه قُلُوبهمْ : يَقُول : أَمَالَ اللَّه قُلُوبهمْ عَنْهُ ; وَقَدْ : 26387 -حَدَّثَنِي يَعْقُوب بْن إِبْرَاهِيم , قَالَ : ثنا هُشَيْم , قَالَ : أَخْبَرَنَا الْعَوَّام , قَالَ : ثنا أَبُو غَالِب , عَنْ أَبِي أُمَامَة فِي قَوْله : { فَلَمَّا زَاغُوا أَزَاغَ اللَّه قُلُوبهمْ } قَالَ : هُمْ الْخَوَارِج .

قُلُوبَهُمْ وَاللَّهُ لَا يَهْدِي الْقَوْمَ

{ وَاَللَّه لَا يَهْدِي الْقَوْم الْفَاسِقِينَ } يَقُول : وَاَللَّه لَا يُوَفِّق لِإِصَابَةِ الْحَقّ الْقَوْم الَّذِينَ اِخْتَارُوا الْكُفْر عَلَى الْإِيمَان .
