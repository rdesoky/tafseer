سَبَّحَ لِلَّهِ مَا فِي السَّمَاوَاتِ وَمَا فِي الْأَرْضِ وَهُوَ الْعَزِيزُ الْحَكِيمُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { سَبَّحَ لِلَّهِ مَا فِي السَّمَوَات وَمَا فِي الْأَرْض وَهُوَ الْعَزِيز الْحَكِيم } يَقُول جَلَّ ثَنَاؤُهُ : { سَبَّحَ لِلَّهِ مَا فِي السَّمَوَات } السَّبْع { وَمَا فِي الْأَرْض } مِنْ الْخَلْق , مُذْعِنِينَ لَهُ بِالْأُلُوهَةِ وَالرُّبُوبِيَّة { وَهُوَ الْعَزِيز } فِي نِقْمَته مِمَّنْ عَصَاهُ مِنْهُمْ , فَكَفَرَ بِهِ , وَخَالَفَ أَمْره { الْحَكِيم } فِي تَدْبِيره إِيَّاهُمْ .
