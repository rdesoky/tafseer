يَا أَيُّهَا الَّذِينَ آمَنُوا هَلْ أَدُلُّكُمْ عَلَى تِجَارَةٍ تُنْجِيكُمْ مِنْ عَذَابٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { يَا أَيّهَا الَّذِينَ آمَنُوا هَلْ أَدُلّكُمْ عَلَى تِجَارَة تُنْجِيكُمْ مِنْ عَذَاب أَلِيم } يَقُول تَعَالَى ذِكْره : { يَا أَيّهَا الَّذِينَ آمَنُوا هَلْ أَدُلّكُمْ عَلَى تِجَارَة تُنْجِيكُمْ مِنْ عَذَاب أَلِيم } مُوجِع , وَذَلِكَ عَذَاب جَهَنَّم ;
