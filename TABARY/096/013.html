أَرَأَيْتَ إِنْ كَذَّبَ وَتَوَلَّى

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { أَرَأَيْت إِنْ كَذَّبَ وَتَوَلَّى } يَقُول تَعَالَى ذِكْره : { أَرَأَيْت إِنْ كَذَّبَ } أَبُو جَهْل بِالْحَقِّ الَّذِي بَعَثَ بِهِ مُحَمَّدًا { وَتَوَلَّى } يَقُول : وَأَدْبَرَ عَنْهُ , فَلَمْ يُصَدِّق بِهِ . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 29165 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { أَرَأَيْت إِنْ كَذَّبَ وَتَوَلَّى } يَعْنِي : أَبَا جَهْل .
