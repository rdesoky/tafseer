مَتَاعٌ فِي الدُّنْيَا

يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { قُلْ } يَا مُحَمَّد لَهُمْ { إِنَّ الَّذِينَ يَفْتَرُونَ عَلَى اللَّه الْكَذِب } فَيَقُولُونَ عَلَيْهِ الْبَاطِل , وَيَدَّعُونَ لَهُ وَلَدًا ; { لَا يُفْلِحُونَ } يَقُول : لَا يَبْقَوْنَ فِي الدُّنْيَا , وَلَكِنْ لَهُمْ { مَتَاع فِي الدُّنْيَا } يُمَتَّعُونَ بِهِ , وَبَلَاغ يَتَبَلَّغُونَ بِهِ إِلَى الْأَجَل الَّذِي كُتِبَ فِنَاؤُهُمْ فِيهِ . وَرُفِعَ قَوْله : { مَتَاع } بِمُضْمِرٍ قَبْلَهُ إِمَّا " ذَلِكَ " وَ إِمَّا " هَذَا " .

ثُمَّ إِلَيْنَا مَرْجِعُهُمْ ثُمَّ نُذِيقُهُمُ الْعَذَابَ الشَّدِيدَ بِمَا كَانُوا يَكْفُرُونَ

{ ثُمَّ إِلَيْنَا مَرْجِعهمْ } يَقُول : ثُمَّ إِذَا اِنْقَضَى أَجَلهمْ الَّذِي كُتِبَ لَهُمْ إِلَيْنَا مَصِيرهمْ وَمُنْقَلَبهمْ . { ثُمَّ نُذِيقهُمْ الْعَذَابَ الشَّدِيدَ } وَذَلِكَ إِصْلَاؤُهُمْ جَهَنَّم ; { بِمَا كَانُوا يَكْفُرُونَ } بِاَللَّهِ فِي الدُّنْيَا , فَيُكَذِّبُونَ رُسُلَهُ وَيَجْحَدُونَ آيَاته .
