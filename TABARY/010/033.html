كَذَلِكَ حَقَّتْ كَلِمَةُ رَبِّكَ عَلَى الَّذِينَ فَسَقُوا أَنَّهُمْ لَا يُؤْمِنُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { كَذَلِكَ حَقَّتْ كَلِمَةُ رَبّك عَلَى الَّذِينَ فَسَقُوا أَنَّهُمْ لَا يُؤْمِنُونَ } يَقُول تَعَالَى ذِكْره : كَمَا قَدْ صُرِفَ هَؤُلَاءِ الْمُشْرِكُونَ عَنْ الْحَقّ إِلَى الضَّلَال , { كَذَلِكَ حَقَّتْ كَلِمَة رَبّك } يَقُول : وَجَبَ عَلَيْهِمْ قَضَاؤُهُ وَحُكْمه فِي السَّابِق مِنْ عِلْمه , { عَلَى الَّذِينَ فَسَقُوا } فَخَرَجُوا مِنْ طَاعَة رَبّهمْ إِلَى مَعْصِيَته وَكَفَرُوا بِهِ ; { أَنَّهُمْ لَا يُؤْمِنُونَ } يَقُول : لَا يُصَدِّقُونَ بِوَحْدَانِيَّةِ اللَّه وَلَا بِنُبُوَّةِ نَبِيّه صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ .
