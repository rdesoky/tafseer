قُلْ إِنَّ الَّذِينَ يَفْتَرُونَ عَلَى اللَّهِ الْكَذِبَ لَا يُفْلِحُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قُلْ إِنَّ الَّذِينَ يَفْتَرُونَ عَلَى اللَّه الْكَذِب لَا يُفْلِحُونَ مَتَاع فِي الدُّنْيَا ثُمَّ إِلَيْنَا مَرْجِعهمْ ثُمَّ نُذِيقهُمْ الْعَذَابَ الشَّدِيدَ بِمَا كَانُوا يَكْفُرُونَ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { قُلْ } يَا مُحَمَّد لَهُمْ { إِنَّ الَّذِينَ يَفْتَرُونَ عَلَى اللَّه الْكَذِب } فَيَقُولُونَ عَلَيْهِ الْبَاطِلَ , وَيَدَّعُونَ لَهُ وَلَدًا ; { لَا يُفْلِحُونَ } يَقُول : لَا يَبْقَوْنَ فِي الدُّنْيَا , وَلَكِنْ لَهُمْ { مَتَاع فِي الدُّنْيَا } يُمَتَّعُونَ بِهِ , وَبَلَاغ يَتَبَلَّغُونَ بِهِ إِلَى الْأَجَل الَّذِي كُتِبَ فَنَاؤُهُمْ فِيهِ . وَرُفِعَ قَوْله : { مَتَاع } بِمُضْمِرٍ قَبْلَهُ إِمَّا " ذَلِكَ " و إِمَّا " هَذَا " .
