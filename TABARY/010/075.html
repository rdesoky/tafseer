ثُمَّ بَعَثْنَا مِنْ بَعْدِهِمْ مُوسَى وَهَارُونَ إِلَى فِرْعَوْنَ وَمَلَئِهِ بِآيَاتِنَا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { ثُمَّ بَعَثْنَا مِنْ بَعْدهمْ مُوسَى وَهَارُون إِلَى فِرْعَوْن وَمَلَئِهِ بِآيَاتِنَا } يَقُول تَعَالَى ذِكْره : ثُمَّ بَعَثْنَا مِنْ بَعْد هَؤُلَاءِ الرُّسُل الَّذِينَ أَرْسَلْنَاهُمْ مِنْ بَعْد نُوح إِلَى قَوْمهمْ مُوسَى وَهَارُون اِبْنَيْ عِمْرَانَ إِلَى فِرْعَوْن مِصْر { وَمَلَئِهِ } , يَعْنِي : وَأَشْرَاف قَوْمه وَسَادَتهمْ , { بِآيَاتِنَا } يَقُول : بِأَدِلَّتِنَا عَلَى حَقِيقَة مَا دَعَوْهُمْ إِلَيْهِ مِنْ الْإِذْعَان لِلَّهِ بِالْعُبُودَةِ , وَالْإِقْرَار لَهُمَا بِالرِّسَالَةِ .

فَاسْتَكْبَرُوا

{ فَاسْتَكْبَرُوا } يَقُول : فَاسْتَكْبَرُوا عَنْ الْإِقْرَار بِمَا دَعَاهُمْ إِلَيْهِ مُوسَى وَهَارُون .

وَكَانُوا قَوْمًا مُجْرِمِينَ

{ وَكَانُوا قَوْمًا مُجْرِمِينَ } يَعْنِي : آثِمِينَ بِرَبِّهِمْ بِكُفْرِهِمْ بِاَللَّهِ تَعَالَى .
