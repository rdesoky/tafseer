وَأَنَّا لَا نَدْرِي أَشَرٌّ أُرِيدَ بِمَنْ فِي الْأَرْضِ أَمْ أَرَادَ بِهِمْ رَبُّهُمْ رَشَدًا

وَقَوْله : { وَأَنَّا لَا نَدْرِي أَشَرّ أُرِيدَ بِمَنْ فِي الْأَرْض أَمْ أَرَادَ بِهِمْ رَبّهمْ رَشَدًا } يَقُول عَزَّ وَجَلَّ مُخْبِرًا عَنْ قِيل هَؤُلَاءِ النَّفَر مِنَ الْجِنّ : وَأَنَّا لَا نَدْرِي أَعَذَابًا أَرَادَ اللَّه أَنْ يُنْزِلهُ بِأَهْلِ الْأَرْض , بِمَنْعِهِ إِيَّانَا السَّمْع مِنَ السَّمَاء وَرَجْمه مَنِ اسْتَمَعَ مِنَّا فِيهَا بِالشُّهُبِ { أَمْ أَرَادَ بِهِمْ رَبّهمْ رَشَدًا } يَقُول : أَمْ أَرَادَ بِهِمْ رَبّهمْ الْهُدَى بِأَنْ يَبْعَث مِنْهُمْ رَسُولًا مُرْشِدًا يُرْشِدهُمْ إِلَى الْحَقّ . وَهَذَا التَّأْوِيل عَلَى التَّأْوِيل الَّذِي ذَكَرْنَاهُ عَنِ ابْن زَيْد قَبْل. وَذُكِرَ عَنِ الْكَلْبِيّ فِي ذَلِكَ مَا : 27201 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , عَنِ الْكَلْبِيّ فِي قَوْله : { وَأَنَّا لَا نَدْرِي أَشَرّ أُرِيدَ بِمَنْ فِي الْأَرْض أَمْ أَرَادَ بِهِمْ رَبّهمْ رَشَدًا } أَنْ يُطِيعُوا هَذَا الرَّسُول فَيُرْشِدهُمْ أَوْ يَعْصُوهُ فَيُهْلِكهُمْ . وَإِنَّمَا قُلْنَا الْقَوْل الْأَوَّل لِأَنَّ قَوْله : { وَأَنَّا لَا نَدْرِي أَشَرّ أُرِيدَ بِمَنْ فِي الْأَرْض } عَقِيب قَوْله : { وَأَنَّا كُنَّا نَقْعُد مِنْهَا مَقَاعِد لِلسَّمْعِ } . .. الْآيَة , فَكَانَ ذَلِكَ بِأَنْ يَكُون مِنْ تَمَام قِصَّة مَا وَلِيَهُ وَقَرُبَ مِنْهُ أَوْلَى بِأَنْ يَكُون مِنْ تَمَام خَبَر مَا بَعُدَ عَنْهُ.
