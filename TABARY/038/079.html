قَالَ رَبِّ فَأَنْظِرْنِي إِلَى يَوْمِ يُبْعَثُونَ

{ قَالَ رَبّ فَأَنْظِرْنِي إِلَى يَوْم يُبْعَثُونَ } يَقُول تَعَالَى ذِكْره : قَالَ إِبْلِيس لِرَبِّهِ : رَبّ فَإِذْ لَعَنْتنِي , وَأَخْرَجْتنِي مِنْ جَنَّتك { فَأَنْظِرْنِي } يَقُول : فَأَخِّرْنِي فِي الْأَجَل , وَلَا تُهْلِكنِي { إِلَى يَوْم يُبْعَثُونَ } يَقُول : إِلَى يَوْم تَبْعَث خَلْقك مِنْ قُبُورهمْ .
