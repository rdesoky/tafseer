إِذْ قَالَ رَبُّكَ لِلْمَلَائِكَةِ إِنِّي خَالِقٌ بَشَرًا مِنْ طِينٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِذْ قَالَ رَبّك لِلْمَلَائِكَةِ إِنِّي خَالِق بَشَرًا مِنْ طِين } وَقَوْله : { إِذْ قَالَ رَبّك } مِنْ صِلَة قَوْله : { إِذْ يَخْتَصِمُونَ } وَتَأْوِيل الْكَلَام : مَا كَانَ لِي مِنْ عِلْم بِالْمَلَأِ الْأَعْلَى إِذْ يَخْتَصِمُونَ حِين قَالَ رَبّك يَا مُحَمَّد { لِلْمَلَائِكَةِ إِنِّي خَالِق بَشَرًا مِنْ طِين } يَعْنِي بِذَلِكَ خُلِقَ آدَم .
