إِنْ هُوَ إِلَّا ذِكْرٌ لِلْعَالَمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنْ هُوَ إِلَّا ذِكْر لِلْعَالَمِينَ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : قُلْ لِهَؤُلَاءِ الْمُشْرِكِينَ مِنْ قَوْمك : { إِنْ هُوَ } يَعْنِي : مَا هَذَا الْقُرْآن { إِلَّا ذِكْر } يَقُول : إِلَّا تَذْكِير مِنْ اللَّه { لِلْعَالَمِينَ } مِنْ الْجِنّ وَالْإِنْس , ذَكَّرَهُمْ رَبّهمْ إِرَادَة اِسْتِنْقَاذ مَنْ آمَنَ بِهِ مِنْهُمْ مِنْ الْهَلَكَة .
