هَذَا وَإِنَّ لِلطَّاغِينَ لَشَرَّ مَآبٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { هَذَا وَإِنَّ لِلطَّاغِينَ لَشَرّ مَآب } يَعْنِي تَعَالَى ذِكْره بِقَوْلِهِ : { هَذَا } : الَّذِي وَصَفْت لِهَؤُلَاءِ الْمُتَّقِينَ : ثُمَّ اِسْتَأْنَفَ جَلَّ وَعَزَّ الْخَبَر عَنْ الْكَافِرِينَ بِهِ الَّذِينَ طَغَوْا عَلَيْهِ وَبَغَوْا , فَقَالَ : { وَإِنَّ لِلطَّاغِينَ } وَهُمْ الَّذِينَ تَمَرَّدُوا عَلَى رَبّهمْ , فَعَصَوْا أَمْره مَعَ إِحْسَانه إِلَيْهِمْ { لَشَرّ مَآب } يَقُول : لَشَرّ مَرْجِع وَمَصِير يَصِيرُونَ إِلَيْهِ فِي الْآخِرَة بَعْد خُرُوجهمْ مِنْ الدُّنْيَا , كَمَا : 23064 - حَدَّثَنَا مُحَمَّد بْن الْحُسَيْن , قَالَ : ثنا أَحْمَد بْن الْمُفَضَّل , قَالَ : ثنا أَسْبَاط , عَنْ السُّدِّيّ { وَإِنَّ لِلطَّاغِينَ لَشَرّ مَآب } قَالَ : لَشَرّ مُنْقَلَب .
