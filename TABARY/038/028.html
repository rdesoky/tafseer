أَمْ نَجْعَلُ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ كَالْمُفْسِدِينَ فِي الْأَرْضِ

وَقَوْله : { أَمْ نَجْعَل الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَات كَالْمُفْسِدِينَ فِي الْأَرْض } يَقُول : أَنَجْعَلُ الَّذِينَ صَدَقُوا اللَّه وَرَسُوله وَعَمِلُوا بِمَا أَمَرَ اللَّه بِهِ , وَانْتَهَوْا عَمَّا نَهَاهُمْ عَنْهُ { كَالْمُفْسِدِينَ فِي الْأَرْض } يَقُول : كَاَلَّذِينَ يُشْرِكُونَ بِاَللَّهِ وَيَعْصُونَهُ وَيُخَالِفُونَ أَمْره وَنَهْيه .

أَمْ نَجْعَلُ الْمُتَّقِينَ كَالْفُجَّارِ

{ أَمْ نَجْعَل الْمُتَّقِينَ } يَقُول : الَّذِينَ اِتَّقُوا اللَّه بِطَاعَتِهِ وَرَاقَبُوهُ , فَحَذِرُوا مَعَاصِيه { كَالْفُجَّارِ } يَعْنِي : كَالْكُفَّارِ الْمُنْتَهِكِينَ حُرُمَات اللَّه .
