إِنَّ عَذَابَ رَبِّكَ لَوَاقِعٌ

وَقَوْله : { إِنَّ عَذَاب رَبّك لَوَاقِع } يَقُول تَعَالَى ذِكْرُهُ لِنَبِيِّهِ صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { إِنَّ عَذَاب رَبّك لَوَاقِع } يَا مُحَمَّد , لَكَائِن حَالّ بِالْكَافِرِينَ بِهِ يَوْم الْقِيَامَة. كَمَا : 25022 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { إِنَّ عَذَاب رَبّك لَوَاقِع } وَقَعَ الْقَسَم هَا هُنَا { إِنَّ عَذَاب رَبّك لَوَاقِع } وَذَلِكَ يَوْم الْقِيَامَة .
