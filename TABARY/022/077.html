يَا أَيُّهَا الَّذِينَ آمَنُوا ارْكَعُوا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { يَا أَيّهَا الَّذِي آمَنُوا ارْكَعُوا وَاسْجُدُوا } يَقُول تَعَالَى ذِكْره : يَا أَيّهَا الَّذِينَ صَدَّقُوا اللَّه وَرَسُوله { ارْكَعُوا } لِلَّهِ فِي صَلَاتكُمْ { وَاسْجُدُوا } لَهُ فِيهَا .

وَاسْجُدُوا وَاعْبُدُوا

يَقُول : وَذِلُّوا لِرَبِّكُمْ , وَاخْضَعُوا لَهُ بِالطَّاعَةِ ,

رَبَّكُمْ وَافْعَلُوا

الَّذِي أَمَرَكُمْ رَبّكُمْ بِفِعْلِهِ ;

الْخَيْرَ لَعَلَّكُمْ

يَقُول : لِتُفْلِحُوا بِذَلِكَ , فَتُدْرِكُوا بِهِ طَلَبَاتكُمْ عِنْد رَبّكُمْ.
