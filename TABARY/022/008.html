وَمِنَ النَّاسِ مَنْ يُجَادِلُ فِي اللَّهِ بِغَيْرِ عِلْمٍ وَلَا هُدًى وَلَا كِتَابٍ مُنِيرٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَمِنَ النَّاس مَنْ يُجَادِل فِي اللَّه بِغَيْرِ عِلْم وَلَا هُدًى وَلَا كِتَاب مُنِير } يَقُول تَعَالَى ذِكْره : وَمِنَ النَّاس مَنْ يُخَاصِم فِي تَوْحِيد اللَّه وَإِفْرَاده بِالْأُلُوهَةِ بِغَيْرِ عِلْم مِنْهُ بِمَا يُخَاصِم بِهِ . { وَلَا هُدًى } يَقُول : وَبِغَيْرِ بَيَان مَعَهُ لِمَا يَقُول وَلَا بُرْهَان . { وَلَا كِتَاب مُنِير } يَقُول : وَبِغَيْرِ كِتَاب مِنَ اللَّه أَتَاهُ لِصِحَّةِ مَا يَقُول . { مُنِير } يَقُول يُنِير عَنْ حُجَّته , وَإِنَّمَا يَقُول مَا يَقُول مِنَ الْجَهْل ظَنًّا مِنْهُ وَحِسْبَانًا . وَذَكَرَ أَنْ عُنِيَ بِهَذِهِ الْآيَة وَالَّتِي بَعْدهَا النَّضْر بْن الْحَارِث مِنْ بَنِي عَبْد الدَّار .
