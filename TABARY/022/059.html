لَيُدْخِلَنَّهُمْ مُدْخَلًا يَرْضَوْنَهُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { لَيُدْخِلَنَّهُمْ مُدْخَلًا يَرْضَوْنَهُ } يَقُول تَعَالَى ذِكْره : لَيُدْخِلَنَّ اللَّه الْمَقْتُول فِي سَبِيله مِنَ الْمُهَاجِرِينَ وَالْمَيِّت مِنْهُمْ { مُدْخَلًا يَرْضَوْنَهُ } وَذَلِكَ الْمُدْخَل هُوَ الْجَنَّة .

وَإِنَّ اللَّهَ لَعَلِيمٌ حَلِيمٌ

{ لَعَلِيم } بِمَنْ يُهَاجِر فِي سَبِيله مِمَّنْ يَخْرُج مِنْ دَاره طَلَب الْغَنِيمَة أَوْ عَرَض مِنْ عُرُوض الدُّنْيَا . { حَلِيم } عَنْ عُصَاة خَلْقه , بِتَرْكِهِ مُعَاجَلَتهمْ بِالْعُقُوبَةِ وَالْعَذَاب .
