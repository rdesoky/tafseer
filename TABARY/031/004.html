الَّذِينَ يُقِيمُونَ الصَّلَاةَ

{ الَّذِينَ يُقِيمُونَ الصَّلَاةَ } يَقُول : الَّذِينَ يُقِيمُونَ الصَّلَاةَ الْمَفْرُوضَةَ بِحُدُودِهَا

وَيُؤْتُونَ الزَّكَاةَ

{ وَيُؤْتُونَ الزَّكَاةَ } مَنْ جَعَلَهَا اللَّه لَهُ الْمَفْرُوضَةَ فِي أَمْوَالهمْ

وَهُمْ بِالْآخِرَةِ هُمْ يُوقِنُونَ

{ وَهُمْ بِالْآخِرَةِ هُمْ يُوقِنُونَ } يَقُول : يَفْعَلُونَ ذَلِكَ وَهُمْ بِجَزَاءِ اللَّه وَثَوَابه لِمَنْ فَعَلَ ذَلِكَ فِي الْآخِرَة يُوقِنُونَ.
