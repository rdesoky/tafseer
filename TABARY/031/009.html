خَالِدِينَ فِيهَا وَعْدَ اللَّهِ حَقًّا وَهُوَ الْعَزِيزُ الْحَكِيمُ

{ خَالِدِينَ فِيهَا } يَقُول : مَاكِثِينَ فِيهَا إِلَى غَيْر نِهَايَة { وَعْد اللَّه حَقًّا } يَقُول : وَعَدَهُمْ اللَّه وَعْدًا حَقًّا , لَا شَكَّ فِيهِ وَلَا خُلْف لَهُ { وَهُوَ الْعَزِيز } يَقُول : وَهُوَ الشَّدِيد فِي انْتِقَامه مِنْ أَهْل الشِّرْك بِهِ , وَالصَّادِّينَ عَنْ سَبِيله , { الْحَكِيم } فِي تَدْبِير خَلْقه .
