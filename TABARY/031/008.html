إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ لَهُمْ جَنَّاتُ النَّعِيمِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَات لَهُمْ جَنَّات النَّعِيم } يَقُول تَعَالَى ذِكْره : { إِنَّ الَّذِينَ آمَنُوا } بِاللَّهِ فَوَحَّدُوهُ , وَصَدَّقُوا رَسُولَهُ وَاتَّبَعُوهُ { وَعَمِلُوا الصَّالِحَات } يَقُول : فَأَطَاعُوا اللَّه , فَعَمِلُوا بِمَا أَمَرَهُمْ فِي كِتَابه وَعَلَى لِسَان رَسُوله , وَانْتَهَوْا عَمَّا نَهَاهُمْ عَنْهُ { لَهُمْ جَنَّات النَّعِيم } يَقُول : لِهَؤُلَاءِ بَسَاتِين النَّعِيم
