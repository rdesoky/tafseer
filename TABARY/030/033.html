وَإِذَا مَسَّ النَّاسَ ضُرٌّ دَعَوْا رَبَّهُمْ مُنِيبِينَ إِلَيْهِ ثُمَّ إِذَا أَذَاقَهُمْ مِنْهُ رَحْمَةً إِذَا فَرِيقٌ مِنْهُمْ بِرَبِّهِمْ يُشْرِكُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَإِذَا مَسَّ النَّاس ضُرّ دَعَوْا رَبّهمْ مُنِيبِينَ إِلَيْهِ ثُمَّ إِذَا أَذَاقَهُمْ مِنْهُ رَحْمَة إِذَا فَرِيق مِنْهُمْ بِرَبِّهِمْ يُشْرِكُونَ } يَقُول تَعَالَى ذِكْره : وَإِذَا مَسَّ هَؤُلَاءِ الْمُشْرِكِينَ الَّذِينَ يَجْعَلُونَ مَعَ اللَّه إِلَهًا آخَر ضُرّ , فَأَصَابَتْهُمْ شِدَّة وَجَدُوب وَقُحُوط { دَعَوْا رَبّهمْ } يَقُول : أَخْلَصُوا لِرَبِّهِمْ التَّوْحِيد , وَأَفْرَدُوهُ بِالدُّعَاءِ وَالتَّضَرُّع إِلَيْهِ , وَاسْتَغَاثُوا بِهِ مُنِيبِينَ إِلَيْهِ , تَائِبِينَ إِلَيْهِ مِنْ شِرْكهمْ وَكُفْرهمْ { ثُمَّ إِذَا أَذَاقَهُمْ مِنْهُ رَحْمَة } يَقُول : ثُمَّ إِذَا كَشَفَ رَبّهمْ تَعَالَى ذِكْره عَنْهُمْ ذَلِكَ الضُّرّ وَفَرَّجَهُ عَنْهُمْ وَأَصَابَهُمْ بِرَخَاءٍ وَخِصْب وَسَعَة , { إِذَا فَرِيق مِنْهُمْ } يَقُول : إِذَا جَمَاعَة مِنْهُمْ بِرَبِّهِمْ { يُشْرِكُونَ } يَقُول : يَعْبُدُونَ مَعَهُ الْآلِهَة وَالْأَوْثَان.
