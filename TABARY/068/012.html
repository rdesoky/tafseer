مَنَّاعٍ لِلْخَيْرِ

قَوْله : { مَنَّاع لِلْخَيْرِ }. يَقُول تَعَالَى ذِكْره : بَخِيل بِالْمَالِ ضَنِين بِهِ عَنِ الْحُقُوق .

مُعْتَدٍ أَثِيمٍ

وَقَوْله : { مُعْتَدٍ } يَقُول : مُعْتَدٍ عَلَى النَّاس { أَثِيم } : ذِي إِثْم بِرَبِّهِ . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 26809- حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة , فِي قَوْله : { مُعْتَدٍ } فِي عَمَله { أَثِيم } بِرَبِّهِ .
