إِنَّ لِلْمُتَّقِينَ عِنْدَ رَبِّهِمْ جَنَّاتِ النَّعِيمِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّ لِلْمُتَّقِينَ عِنْد رَبّهمْ جَنَّات النَّعِيم } . يَقُول تَعَالَى ذِكْره : { إِنَّ لِلْمُتَّقِينَ } الَّذِينَ اتَّقَوْا عُقُوبَة اللَّه بِأَدَاءِ فَرَائِضه , وَاجْتِنَاب مَعَاصِيه { عِنْد رَبّهمْ جَنَّات النَّعِيم } يَعْنِي : بَسَاتِين النَّعِيم الدَّائِم.
