لَوْلَا أَنْ تَدَارَكَهُ نِعْمَةٌ مِنْ رَبِّهِ لَنُبِذَ بِالْعَرَاءِ وَهُوَ مَذْمُومٌ

وَقَوْله : { لَوْلَا أَنْ تَدَارَكَهُ نِعْمَة مِنْ رَبّه } يَقُول جَلَّ ثَنَاؤُهُ : لَوْلَا أَنْ تَدَارَكَ صَاحِب الْحُوت نِعْمَةٌ مِنْ رَبّه , فَرَحِمَهُ بِهَا , وَتَابَ عَلَيْهِ مِنْ مُغَاضَبَته رَبّه { لَنُبِذَ بِالْعَرَاءِ } وَهُوَ الْفَضَاء مِنَ الْأَرْض : وَمِنْهُ قَوْل بْن جَعْدَة : وَرَفَعْت رِجْلًا لَا أَخَاف عِثَارهَا وَنَبَذْت بِالْبَلَدِ الْعَرَاء ثِيَابِي { وَهُوَ مَذْمُوم } اخْتَلَفَ أَهْل التَّأْوِيل فِي مَعْنَى قَوْله : { وَهُوَ مَذْمُوم } فَقَالَ بَعْضهمْ : مَعْنَاهُ وَهُوَ مُلِيم . ذِكْر مَنْ قَالَ ذَلِكَ : 26901 - حَدَّثَنِي عَلِيّ , قَالَ : ثني أَبُو صَالِح , قَالَ : ثني مُعَاوِيَة , عَنْ عَلِيّ , عَنِ ابْن عَبَّاس , فِي قَوْله : { وَهُوَ مَذْمُوم } يَقُول : وَهُوَ مُلِيم . وَقَالَ آخَرُونَ : بَلْ مَعْنَى ذَلِكَ : وَهُوَ مُذْنِب ذِكْر مَنْ قَالَ ذَلِكَ : 26902 - حَدَّثَنَا ابْن عَبْد الْأَعْلَى , قَالَ : ثنا الْمُعْتَمِر , عَنْ أَبِيهِ عَنْ بَكْر { وَهُوَ مَذْمُوم } قَالَ : هُوَ مُذْنِب .
