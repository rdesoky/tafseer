قَدْ فَرَضَ اللَّهُ لَكُمْ تَحِلَّةَ أَيْمَانِكُمْ وَاللَّهُ مَوْلَاكُمْ وَهُوَ الْعَلِيمُ الْحَكِيمُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَدْ فَرَضَ اللَّه لَكُمْ تَحِلَّة أَيْمَانكُمْ وَاللَّه مَوْلَاكُمْ وَهُوَ الْعَلِيم الْحَكِيم }. يَقُول تَعَالَى ذِكْره : قَدْ بَيَّنَ اللَّه عَزَّ وَجَلَّ لَكُمْ تَحِلَّة أَيْمَانكُمْ , وَحَدَّهَا لَكُمْ أَيّهَا النَّاس { وَاللَّه مَوْلَاكُمْ } يَتَوَلَّاكُمْ بِنَصْرِهِ أَيّهَا الْمُؤْمِنُونَ { وَهُوَ الْعَلِيم } بِمَصَالِحِكُمْ { الْحَكِيم } فِي تَدْبِيره إِيَّاكُمْ , وَصَرْفكُمْ فِيمَا هُوَ أَعْلَم بِهِ .
