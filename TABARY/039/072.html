قِيلَ ادْخُلُوا أَبْوَابَ جَهَنَّمَ خَالِدِينَ فِيهَا فَبِئْسَ مَثْوَى الْمُتَكَبِّرِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قِيلَ اُدْخُلُوا أَبْوَاب جَهَنَّم خَالِدِينَ فِيهَا فَبِئْسَ مَثْوَى الْمُتَكَبِّرِينَ } يَقُول تَعَالَى ذِكْره : فَتَقُول خَزَنَة جَهَنَّم لِلَّذِينَ كَفَرُوا حِينَئِذٍ : { اُدْخُلُوا أَبْوَاب جَهَنَّم } السَّبْعَة عَلَى قَدْر مَنَازِلكُمْ فِيهَا { خَالِدِينَ فِيهَا } يَقُول : مَاكِثِينَ فِيهَا لَا يُنْقَلُونَ عَنْهَا إِلَى غَيْرهَا . { فَبِئْسَ مَثْوَى الْمُتَكَبِّرِينَ } يَقُول : فَبِئْسَ مَسْكَن الْمُتَكَبِّرِينَ عَلَى اللَّه فِي الدُّنْيَا , أَنْ يُوَحِّدُوهُ وَيُفْرِدُوا لَهُ الْأُلُوهَة , جَهَنَّم يَوْم الْقِيَامَة .
