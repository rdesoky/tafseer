وَلَقَدْ آتَيْنَا بَنِي إِسْرَائِيلَ الْكِتَابَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَقَدْ آتَيْنَا بَنِي إِسْرَائِيل الْكِتَاب } يَقُول تَعَالَى ذِكْره : { وَلَقَدْ آتَيْنَا } يَا مُحَمَّد { بَنِي إِسْرَائِيل الْكِتَاب } يَعْنِي التَّوْرَاة وَالْإِنْجِيل .

وَالْحُكْمَ

{ وَالْحُكْم } يَعْنِي الْفَهْم بِالْكِتَابِ , وَالْعِلْم بِالسُّنَنِ الَّتِي لَمْ تَنْزِل فِي الْكِتَاب .

وَالنُّبُوَّةَ

{ وَالنُّبُوَّة } يَقُول : وَجَعَلْنَا مِنْهُمْ أَنْبِيَاء وَرُسُلًا إِلَى الْخَلْق .

وَرَزَقْنَاهُمْ مِنَ الطَّيِّبَاتِ

{ وَرَزَقْنَاهُمْ مِنَ الطَّيِّبَات } يَقُول : وَأَطْعَمْنَاهُمْ مِنْ طَيِّبَات أَرْزَاقنَا , وَذَلِكَ مَا أَطْعَمَهُمْ مِنْ الْمَنّ وَالسَّلْوَى

وَفَضَّلْنَاهُمْ عَلَى الْعَالَمِينَ

{ وَفَضَّلْنَاهُمْ عَلَى الْعَالَمِينَ } يَقُول : وَفَضَّلْنَاهُمْ عَلَى عَالَمِي أَهْل زَمَانهمْ فِي أَيَّام فِرْعَوْن وَعَهْده فِي نَاحِيَتهمْ بِمِصْرَ وَالشَّام .
