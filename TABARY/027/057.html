فَأَنْجَيْنَاهُ وَأَهْلَهُ إِلَّا امْرَأَتَهُ قَدَّرْنَاهَا مِنَ الْغَابِرِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَأَنْجَيْنَاهُ وَأَهْله إِلَّا اِمْرَأَته قَدَّرْنَاهَا مِنْ الْغَابِرِينَ } يَقُول تَعَالَى ذِكْره : فَأَنْجَيْنَا لُوطًا وَأَهْله سِوَى اِمْرَأَته مِنْ عَذَابنَا حِين أَحْلَلْنَاهُ بِهِمْ , ثُمَّ { قَدَّرْنَاهَا } يَقُول : فَإِنَّ اِمْرَأَته قَدَّرْنَاهَا : جَعَلْنَاهَا بِتَقْدِيرِنَا { مِنْ الْغَابِرِينَ } مِنْ الْبَاقِينَ .
