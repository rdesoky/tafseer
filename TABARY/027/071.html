وَيَقُولُونَ مَتَى هَذَا الْوَعْدُ إِنْ كُنْتُمْ صَادِقِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَيَقُولُونَ مَتَى هَذَا الْوَعْد إِنْ كُنْتُمْ صَادِقِينَ } يَقُول تَعَالَى ذِكْره : وَيَقُول مُشْرِكُو قَوْمك يَا مُحَمَّد , الْمُكَذِّبُوك فِيمَا أَتَيْتهمْ بِهِ مِنْ عِنْد رَبّك . { مَتَى } يَكُون { هَذَا الْوَعْد } الَّذِي تَعِدنَاهُ مِنْ الْعَذَاب , الَّذِي هُوَ بِنَا فِيمَا تَقُول حَالّ { إِنْ كُنْتُمْ صَادِقِينَ } فِيمَا تَعِدُونَنَا بِهِ .
