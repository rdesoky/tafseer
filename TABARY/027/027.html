قَالَ سَنَنْظُرُ أَصَدَقْتَ أَمْ كُنْتَ مِنَ الْكَاذِبِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ سَنَنْظُرُ أَصَدَقْت أَمْ كُنْت مِنْ الْكَاذِبِينَ } يَقُول تَعَالَى ذِكْره : { قَالَ } سُلَيْمَان لِلْهُدْهُدِ : { سَنَنْظُرُ } فِيمَا اِعْتَذَرْت بِهِ مِنْ الْعُذْر , وَاحْتَجَجْت بِهِ مِنْ الْحُجَّة لِغَيْبَتِك عَنَّا , وَفِيمَا جِئْتنَا بِهِ مِنْ الْخَيْر { أَصَدَقْت } فِي ذَلِكَ كُلّه { أَمْ كُنْت مِنْ الْكَاذِبِينَ } فِيهِ
