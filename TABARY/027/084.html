حَتَّى إِذَا جَاءُوا قَالَ أَكَذَّبْتُمْ بِآيَاتِي وَلَمْ تُحِيطُوا بِهَا عِلْمًا أَمْ مَاذَا كُنْتُمْ

وَقَوْله : { حَتَّى إِذَا جَاءُوا قَالَ أَكَذَّبْتُمْ بِآيَاتِي } يَقُول تَعَالَى ذِكْره : حَتَّى إِذَا جَاءَ مِنْ كُلّ أُمَّة فَوْج مِمَّنْ يُكَذِّب بِآيَاتِنَا فَاجْتَمَعُوا قَالَ اللَّه : { أَكَذَّبْتُمْ بِآيَاتِي } أَيْ بِحُجَجِي وَأَدِلَّتِي { وَلَمْ تُحِيطُوا بِهَا عِلْمًا } يَقُول وَلَمْ تَعْرِفُوهَا حَقّ مَعْرِفَتهَا ؟ { أَمْ مَاذَا كُنْتُمْ تَعْمَلُونَ } فِيهَا مِنْ تَكْذِيب أَوْ تَصْدِيق .
