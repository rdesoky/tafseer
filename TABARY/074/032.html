كَلَّا وَالْقَمَرِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { كَلَّا وَالْقَمَر } يَعْنِي تَعَالَى ذِكْره بِقَوْلِهِ { كَلَّا } لَيْسَ الْقَوْل كَمَا يَقُول مَنْ زَعَمَ أَنَّهُ يَكْفِي أَصْحَابه الْمُشْرِكِينَ خَزَنَة جَهَنَّم حَتَّى يُجْهِضهُمْ عَنْهَا , ثُمَّ أَقْسَمَ رَبّنَا تَعَالَى فَقَالَ : { وَالْقَمَر }
