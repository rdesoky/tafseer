اللَّهُ الَّذِي جَعَلَ لَكُمُ الْأَنْعَامَ لِتَرْكَبُوا مِنْهَا وَمِنْهَا تَأْكُلُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { اللَّه الَّذِي جَعَلَ لَكُمْ الْأَنْعَام لِتَرْكَبُوا مِنْهَا وَمِنْهَا تَأْكُلُونَ } يَقُول تَعَالَى ذِكْره : { اللَّه } الَّذِي لَا تَصْلُح الْأُلُوهَة إِلَّا لَهُ أَيّهَا الْمُشْرِكُونَ بِهِ مِنْ قُرَيْش { الَّذِي جَعَلَ لَكُمْ الْأَنْعَام } مِنْ الْإِبِل وَالْبَقَر وَالْغَنَم وَالْخَيْل , وَغَيْر ذَلِكَ مِنْ الْبَهَائِم الَّتِي يَقْتَنِيهَا أَهْل الْإِسْلَام لِمَرْكَبٍ أَوْ لِمَطْعَمٍ { لِتَرْكَبُوا مِنْهَا } يَعْنِي : الْخَيْل وَالْحَمِير { وَمِنْهَا تَأْكُلُونَ } يَعْنِي الْإِبِل وَالْبَقَر وَالْغَنَم . وَقَالَ : { لِتَرْكَبُوا مِنْهَا } وَمَعْنَاهُ : لِتَرْكَبُوا مِنْهَا بَعْضًا وَمِنْهَا بَعْضًا تَأْكُلُونَ , فَحُذِفَ اِسْتِغْنَاء بِدَلَالَةِ الْكَلَام عَلَى مَا حُذِفَ .
