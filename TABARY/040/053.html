وَلَقَدْ آتَيْنَا مُوسَى الْهُدَى وَأَوْرَثْنَا بَنِي إِسْرَائِيلَ الْكِتَابَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَقَدْ آتَيْنَا مُوسَى الْهُدَى وَأَوْرَثْنَا بَنِي إِسْرَائِيل الْكِتَاب } يَقُول تَعَالَى ذِكْره { وَلَقَدْ آتَيْنَا مُوسَى } الْبَيَان لِلْحَقِّ الَّذِي بَعَثْنَاهُ بِهِ كَمَا آتَيْنَا ذَلِكَ مُحَمَّدًا فَكَذَّبَ بِهِ فِرْعَوْن وَقَوْمه , كَمَا كَذَّبَتْ قُرَيْش مُحَمَّدًا { وَأَوْرَثْنَا بَنِي إِسْرَائِيل الْكِتَاب } يَقُول : وَأَوْرَثْنَا بَنِي إِسْرَائِيل التَّوْرَاة , فَعَلَّمْنَاهُمُوهَا , وَأَنْزَلْنَا إِلَيْهِمْ { هُدًى } يَعْنِي بَيَانًا لِأَمْرِ دِينهمْ , وَمَا أَلْزَمْنَاهُمْ مِنْ فَرَائِضهَا , { وَذِكْرَى لِأُولِي الْأَلْبَاب } يَقُول : وَتَذْكِيرًا مِنَّا لِأَهْلِ الْحِجَا وَالْعُقُول مِنْهُمْ بِهَا .
