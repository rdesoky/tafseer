هُوَ الَّذِي يُحْيِي وَيُمِيتُ فَإِذَا قَضَى أَمْرًا فَإِنَّمَا يَقُولُ لَهُ كُنْ فَيَكُونُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { هُوَ الَّذِي يُحْيِ وَيُمِيت فَإِذَا قَضَى أَمْرًا فَإِنَّمَا يَقُول لَهُ كُنْ فَيَكُون } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : قُلْ لَهُمْ يَا مُحَمَّد : { هُوَ الَّذِي يُحْيِي وَيُمِيت } يَقُول قُلْ لَهُمْ : وَمِنْ صِفَته جَلَّ ثَنَاؤُهُ أَنَّهُ هُوَ الَّذِي يُحْيِي مَنْ يَشَاء بَعْد مَمَاته , وَيُمِيت مَنْ يَشَاء مِنْ الْأَحْيَاء بَعْد حَيَاته و { إِذَا قَضَى أَمْرًا } يَقُول : وَإِذَا قَضَى كَوْن أَمْر مِنْ الْأُمُور الَّتِي يُرِيد تَكْوِينَهَا { فَإِنَّمَا يَقُول لَهُ كُنْ } يَعْنِي لِلَّذِي يُرِيد تَكْوِينَهُ كُنْ , فَيَكُون مَا أَرَادَ تَكْوِينَهُ مَوْجُودًا بِغَيْرِ مُعَانَاة , وَلَا كُلْفَة مُؤْنَة .
