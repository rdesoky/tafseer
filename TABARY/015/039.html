قَالَ رَبِّ بِمَا أَغْوَيْتَنِي لَأُزَيِّنَنَّ لَهُمْ فِي الْأَرْضِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ رَبّ بِمَا أَغْوَيْتنِي لَأُزَيِّنَنَّ لَهُمْ فِي الْأَرْض } يَقُول ـ تَعَالَى ذِكْره ـ : قَالَ إِبْلِيس : { رَبّ بِمَا أَغْوَيْتنِي } بِإِغْوَائِك { لَأُزَيِّنَنَّ لَهُمْ فِي الْأَرْض } وَكَأَنَّ قَوْله : { بِمَا أَغْوَيْتنِي } خَرَجَ مَخْرَج الْقَسَم , كَمَا يُقَال : بِاَللَّهِ , أَوْ بِعِزَّةِ اللَّه لَأُغْوِيَنَّهُمْ. وَعَنَى بِقَوْلِهِ : { لَأُزَيِّنَنَّ لَهُمْ فِي الْأَرْض } لَأَحَسِّنَنَّ لَهُمْ مَعَاصِيك , وَلَأُحَبِّبَنَّهَا إِلَيْهِمْ فِي الْأَرْض .

وَلَأُغْوِيَنَّهُمْ أَجْمَعِينَ

{ وَلَأُغْوِيَنَّهُم أَجْمَعِينَ } يَقُول : وَلَأُضِلَّنَّهُمْ عَنْ سَبِيل الرَّشَاد.
