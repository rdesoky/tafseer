وَإِنَّا لَنَحْنُ نُحْيِي

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَإِنَّا لَنَحْنُ نُحْيِي } يَقُول ـ تَعَالَى ذِكْره ـ : { وَإِنَّا لَنَحْنُ نُحْيِي } مَنْ كَانَ مَيْتًا إِذَا أَرَدْنَا

وَنُمِيتُ

{ وَنُمِيت } مَنْ كَانَ حَيًّا إِذَا شِئْنَا .

وَنَحْنُ الْوَارِثُونَ

{ وَنَحْنُ الْوَارِثُونَ } يَقُول : وَنَحْنُ نَرِث الْأَرْض وَمَنْ عَلَيْهَا بِأَنْ نُمِيت جَمِيعهمْ , فَلَا يَبْقَى حَيّ سِوَانَا إِذَا جَاءَ ذَلِكَ الْأَجَل.
