وَلَا يَتَمَنَّوْنَهُ أَبَدًا بِمَا قَدَّمَتْ أَيْدِيهِمْ وَاللَّهُ عَلِيمٌ بِالظَّالِمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَا يَتَمَنَّوْنَهُ أَبَدًا بِمَا قَدَّمَتْ أَيْدِيهمْ وَاَللَّه عَلِيم بِالظَّالِمِينَ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { وَلَا يَتَمَنَّوْنَهُ أَبَدًا } يَقُول : وَلَا يَتَمَنَّى الْيَهُود الْمَوْت أَبَدًا { بِمَا قَدَّمَتْ أَيْدِيهمْ } يَعْنِي : بِمَا اِكْتَسَبُوا فِي هَذِهِ الدُّنْيَا مِنْ الْآثَام , وَاجْتَرَحُوا مِنْ السَّيِّئَات { وَاَللَّه عَلِيم بِالظَّالِمِينَ } يَقُول : وَاَللَّه ذُو عِلْم بِمَنْ ظَلَمَ مِنْ خَلْقه نَفْسه , فَأَوْبَقَهَا بِكُفْرِهِ بِاَللَّهِ .
