تَبَارَكَ اسْمُ رَبِّكَ ذِي الْجَلَالِ وَالْإِكْرَامِ

وَقَوْله : { تَبَارَكَ اسْم رَبّك } يَقُول تَعَالَى ذِكْرُهُ : تَبَارَكَ ذِكْر رَبّك يَا مُحَمَّد { ذِي الْجَلَال } يَعْنِي ذِي الْعَظَمَة { وَالْإِكْرَام } يَعْنِي : وَمَنْ لَهُ الْإِكْرَام مِنْ جَمِيع خَلْقه . كَمَا : 25740 - حَدَّثَنِي عَلِيّ , قَالَ : ثنا أَبُو صَالِح , قَالَ : ثني مُعَاوِيَة , عَنْ عَلِيّ , عَنْ ابْن عَبَّاس قَوْله : { ذِي الْجَلَال وَالْإِكْرَام } يَقُول : ذُو الْعَظَمَة
