وَمَا تَشَاءُونَ إِلَّا أَنْ يَشَاءَ اللَّهُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَمَا تَشَاءُونَ إِلَّا أَنْ يَشَاء اللَّه } يَقُول تَعَالَى ذِكْره : { وَمَا تَشَاءُونَ } اِتِّخَاذ السَّبِيل إِلَى رَبّكُمْ أَيّهَا النَّاس { إِلَّا أَنْ يَشَاء اللَّه } ذَلِكَ لَكُمْ لِأَنَّ الْأَمْر إِلَيْهِ لَا إِلَيْكُمْ ; وَهُوَ فِي قِرَاءَة عَبْد اللَّه فِيمَا ذُكِرَ : " وَمَا تَشَاءُونَ إِلَّا مَا شَاءَ اللَّه " .

إِنَّ اللَّهَ كَانَ عَلِيمًا حَكِيمًا

وَقَوْله { إِنَّ اللَّه كَانَ عَلِيمًا حَكِيمًا } فَلَنْ يَعْدُو مِنْكُمْ أَحَد مَا سَبَقَ لَهُ فِي عِلْمه بِتَدْبِيرِكُمْ .
