وَأَطِيعُوا اللَّهَ وَأَطِيعُوا الرَّسُولَ فَإِنْ تَوَلَّيْتُمْ فَإِنَّمَا عَلَى رَسُولِنَا الْبَلَاغُ الْمُبِينُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَأَطِيعُوا اللَّه وَأَطِيعُوا الرَّسُول فَإِنْ تَوَلَّيْتُمْ فَإِنَّمَا عَلَى رَسُولنَا الْبَلَاغ الْمُبِين } يَقُول تَعَالَى ذِكْره : { وَأَطِيعُوا اللَّه } أَيّهَا النَّاس فِي أَمْره وَنَهْيه { وَأَطِيعُوا الرَّسُول } صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ { فَإِنْ تَوَلَّيْتُمْ } فَإِنْ أَدْبَرْتُمْ عَنْ طَاعَة اللَّه وَطَاعَة رَسُوله مُسْتَكْبِرِينَ عَنْهَا , فَلَمْ تُطِيعُوا اللَّه وَلَا رَسُوله { فَإِنَّمَا } فَلَيْسَ { عَلَى رَسُولنَا } مُحَمَّد إِلَّا { الْبَلَاغ الْمُبِين } أَنَّهُ بَلَاغ إِلَيْكُمْ لِمَا أَرْسَلْته بِهِ يَقُول جَلَّ ثَنَاؤُهُ : فَقَدْ أَعْذَرَ إِلَيْكُمْ بِالْإِبْلَاغِ وَاَللَّه وَلِيّ الِانْتِقَام مِمَّنْ عَصَاهُ , وَخَالَفَ أَمْره , وَتَوَلَّى عَنْهُ.
