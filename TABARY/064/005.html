أَلَمْ يَأْتِكُمْ نَبَأُ الَّذِينَ كَفَرُوا مِنْ قَبْلُ فَذَاقُوا وَبَالَ أَمْرِهِمْ وَلَهُمْ عَذَابٌ أَلِيمٌ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { أَلَمْ يَأْتِكُمْ نَبَأ الَّذِينَ كَفَرُوا مِنْ قَبْل فَذَاقُوا وَبَال أَمْرهمْ وَلَهُمْ عَذَاب أَلِيم } يَقُول تَعَالَى ذِكْره لِمُشْرِكِي قُرَيْش : أَلَمْ يَأْتِكُمْ أَيّهَا النَّاس خَبَر الَّذِينَ كَفَرُوا مِنْ قَبْلكُمْ , وَذَلِكَ كَقَوْمِ نُوح وَعَاد وَثَمُود وَقَوْم إِبْرَاهِيم وَقَوْم لُوط } { فَذَاقُوا وَبَال أَمْرهمْ } فَمَسَّهُمْ عَذَاب اللَّه إِيَّاهُمْ عَلَى كُفْرهمْ { وَلَهُمْ عَذَاب أَلِيم } يَقُول : وَلَهُمْ عَذَاب مُؤْلِم مُوجِع يَوْم الْقِيَامَة فِي نَار جَهَنَّم , مَعَ الَّذِي أَذَاقَهُمْ اللَّه فِي الدُّنْيَا وَبَال كُفْرهمْ.
