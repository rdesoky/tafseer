قُلْ كُلٌّ مُتَرَبِّصٌ فَتَرَبَّصُوا فَسَتَعْلَمُونَ مَنْ أَصْحَابُ الصِّرَاطِ السَّوِيِّ وَمَنِ اهْتَدَى

الْقَوْل في تَأْويل قَوْله تَعَالَى : { قُلْ كُلّ مُتَرَبّص فَتَرَبَّصُوا فَسَتَعْلَمُونَ مَنْ أَصْحَاب الصّرَاط السَّويّ وَمَنْ اهْتَدَى } . يَقُول تَعَالَى ذكْره لنَبيّه مُحَمَّد صَلَّى اللَّه عَلَيْه وَسَلَّمَ : قُلْ يَا مُحَمَّد : كُلّكُمْ أَيّهَا الْمُشْركُونَ باَللَّه مُتَرَبّص يَقُول : مُنْتَظر لمَنْ يَكُون الْفَلَاح , وَإلَى مَا يَئُول أَمْري وَأَمْركُمْ مُتَوَقّف يَنْتَظر دَوَائر الزَّمَان , { فَتَرَبَّصُوا } يَقُول : فَتَرَقَّبُوا وَانْتَظَرُوا , فَسَتَعْلَمُونَ مَنْ أَهْل الطَّريق الْمُسْتَقيم الْمُعْتَدل الَّذي لَا اعْوجَاج فيه إذَا جَاءَ أَمْر اللَّه وَقَامَتْ الْقيَامَة , أَنَحْنُ أَمْ أَنْتُمْ ؟ { وَمَنْ اهْتَدَى } يَقُول : وَسَتَعْلَمُونَ حينَئذٍ مَنْ الْمُهْتَدي الَّذي هُوَ عَلَى سُنَن الطَّريق الْقَاصد غَيْر الْجَائر عَنْ قَصْده منَّا وَمنْكُمْ . وَفي " مَنْ " منْ قَوْله : { فَسَتَعْلَمُونَ مَنْ أَصْحَاب الصّرَاط السَّويّ } , وَالثَّانيَة منْ قَوْله : { وَمَنْ اهْتَدَى } وَجْهَان : الرَّفْع , وَتَرَكَ إعْمَال تَعْلَمُونَ فيهمَا , كَمَا قَالَ جَلَّ ثَنَاؤُهُ : { لنَعْلَم أَيّ الْحزْبَيْن أَحْصَى } 18 12 وَالنَّصْب عَلَى إعْمَال تَعْلَمُونَ فيهمَا , كَمَا قَالَ جَلَّ ثَنَاؤُهُ : { وَاَللَّه يَعْلَم الْمُفْسد منْ الْمُصْلح } . 2 220
