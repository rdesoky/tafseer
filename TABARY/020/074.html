إِنَّهُ مَنْ يَأْتِ رَبَّهُ مُجْرِمًا فَإِنَّ لَهُ جَهَنَّمَ لَا يَمُوتُ فِيهَا وَلَا يَحْيَا

الْقَوْل في تَأْويل قَوْله تَعَالَى : { إنَّهُ مَنْ يَأْت رَبّه مُجْرمًا فَإنَّ لَهُ جَهَنَّم لَا يَمُوت فيهَا وَلَا يَحْيَى } يَقُول تَعَالَى ذكْره مُخْبرًا عَنْ قيل السَّحَرَة لفرْعَوْن : { إنَّهُ مَنْ يَأْت رَبّه } منْ خَلْقه { مُجْرمًا } يَقُول : مُكْتَسبًا الْكُفْر به , { فَإنَّ لَهُ جَهَنَّم } يَقُول : فَإنَّ لَهُ جَهَنَّم مَأْوَى وَمَسْكَنًا , جَزَاء لَهُ عَلَى كُفْره { لَا يَمُوت فيهَا } فَتَخْرُج نَفْسه { وَلَا يَحْيَا } فَتَسْتَقرّ نَفْسه في مَقَرّهَا فَتَطْمَئنّ , وَلَكنَّهَا تَتَعَلَّق بالْحَنَاجر منْهُمْ .
