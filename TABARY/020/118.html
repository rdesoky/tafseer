إِنَّ لَكَ أَلَّا تَجُوعَ فِيهَا وَلَا تَعْرَى

الْقَوْل في تَأْويل قَوْله تَعَالَى : { إنَّ لَك أَلَّا تَجُوع فيهَا وَلَا تَعْرَى } يَقُول تَعَالَى ذكْره , مُخْبرًا عَنْ قيله لآدَم حين أَسْكَنَهُ الْجَنَّة : { إنَّ لَك } يَا آدَم { أَنْ لَا تَجُوع فيهَا وَلَا تَعْرَى } . و " أَنْ " في قَوْله { أَلَّا تَجُوع فيهَا } في مَوْضع نَصْب بإنَّ الَّتي في قَوْله : { إنَّ لَك } .
