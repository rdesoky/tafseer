جَنَّاتُ عَدْنٍ تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ خَالِدِينَ فِيهَا

الْقَوْل في تَأْويل قَوْله تَعَالَى : { جَنَّات عَدْن تَجْري منْ تَحْتهَا الْأَنْهَار خَالدينَ فيهَا } يَقُول تَعَالَى ذكْره : { وَمَنْ يَأْته مُؤْمنًا قَدْ عَملَ الصَّالحَات فَأُولَئكَ لَهُمْ الدَّرَجَات الْعُلَى } . ثُمَّ بَيَّنَ تلْكَ الدَّرَجَات الْعُلَى مَا هيَ , فَقَالَ : هُنَّ { جَنَّات عَدْن } يَعْني : جَنَّات إقَامَة لَا ظَعْن عَنْهَا وَلَا نَفَاد لَهَا وَلَا فَنَاء { تَجْري منْ تَحْتهَا الْأَنْهَار } يَقُول : تَجْري منْ تَحْت أَشْجَارهَا الْأَنْهَار { خَالدينَ فيهَا } يَقُول : مَاكثينَ فيهَا إلَى غَيْر غَايَة مَحْدُودَة ; فَالْجَنَّات منْ قَوْله { جَنَّات عَدْن } مَرْفُوعَة بالرَّدّ عَلَى الدَّرَجَات , كَمَا : 18271 - حَدَّثَنَا الْقَاسم , قَالَ : ثنا الْحُسَيْن , قَالَ : ثني حَجَّاج , عَنْ ابْن جُرَيْج , في قَوْله : { وَمَنْ يَأْته مُؤْمنًا قَدْ عَملَ الصَّالحَات فَأُولَئكَ لَهُمْ الدَّرَجَات الْعُلَى } قَالَ : عَدْن .

وَذَلِكَ جَزَاءُ مَنْ تَزَكَّى

وَقَوْله : { وَذَلكَ جَزَاء مَنْ تَزَكَّى } يَقُول : وَهَذه الدَّرَجَات الْعُلَى الَّتي هيَ جَنَّات عَدْن عَلَى مَا وَصَفَ جَلَّ جَلَاله ثَوَاب مَنْ تَزَكَّى , يَعْني : مَنْ تَطَهَّرَ منْ الذُّنُوب , فَأَطَاعَ اللَّه فيمَا أَمَرَهُ , وَلَمْ يُدَنّس نَفْسه بمَعْصيَته فيمَا نَهَاهُ عَنْهُ .
