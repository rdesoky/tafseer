يَوْمَئِذٍ لَا تَنْفَعُ الشَّفَاعَةُ إِلَّا مَنْ أَذِنَ لَهُ الرَّحْمَنُ وَرَضِيَ لَهُ قَوْلًا

الْقَوْل في تَأْويل قَوْله تَعَالَى : { يَوْمئذٍ لَا تَنْفَع الشَّفَاعَة إلَّا مَنْ أَذنَ لَهُ الرَّحْمَن وَرَضيَ لَهُ قَوْلًا } يَقُول تَعَالَى ذكْره { يَوْمئذٍ لَا تَنْفَع الشَّفَاعَة إلَّا } شَفَاعَة { مَنْ أَذنَ لَهُ الرَّحْمَن } أَنْ يَشْفَع { وَرَضيَ لَهُ قَوْلًا } وَأَدْخَلَ في الْكَلَام لَهُ دَليلًا عَلَى إضَافَة الْقَوْل إلَى كنَايَة " مَنْ " وَذَلكَ كَقَوْل الْقَائل الْآخَر : رَضيت لَك عَمَلك , وَرَضيته منْك , وَمَوْضع مَنْ منْ قَوْله { إلَّا مَنْ أَذنَ لَهُ } نُصبَ لأَنَّهُ خلَاف الشَّفَاعَة .
