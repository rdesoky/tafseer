وَيَتَجَنَّبُهَا الْأَشْقَى

{ وَيَتَجَنَّبهَا } يَقُول : وَيَتَجَنَّب الذِّكْرَى { الْأَشْقَى } يَعْنِي : أَشْقَى الْفَرِيقَيْنِ { الَّذِي يَصْلَى النَّار الْكُبْرَى } وَهُمْ الَّذِينَ لَمْ تَنْفَعهُمْ الذِّكْرَى . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 28646 - حَدَّثَنَا بِشْر , قَالَ : ثَنَا يَزِيد , قَالَ : ثَنَا سَعِيد , عَنْ قَتَادَة , قَوْله : { فَذَكِّرْ إِنْ نَفَعَتْ الذِّكْرَى سَيَذَّكَّرُ مَنْ يَخْشَى } فَاتَّقُوا اللَّه , مَا خَشِيَ اللَّه عَبْد قَطُّ إِلَّا ذَكَرَهُ { وَيَتَجَنَّبهَا الْأَشْقَى } فَلَا وَاَللَّه لَا يَتَنَكَّب عَبْد هَذَا الذِّكْر زُهْدًا فِيهِ وَبُغْضًا لِأَهْلِهِ , إِلَّا شَقِيّ بَيِّن الشَّقَاء .
