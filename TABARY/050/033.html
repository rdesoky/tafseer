مَنْ خَشِيَ الرَّحْمَنَ بِالْغَيْبِ

وَقَوْله : { مَنْ خَشِيَ الرَّحْمَن بِالْغَيْبِ } يَقُول : مَنْ خَافَ اللَّه فِي الدُّنْيَا مِنْ قَبْل أَنْ يَلْقَاهُ , فَأَطَاعَهُ , وَاتَّبَعَ أَمْره . وَفِي { مَنْ } فِي قَوْله : { مَنْ خَشِيَ } وَجْهَانِ مِنْ الْإِعْرَاب : الْخَفْض عَلَى اِتِّبَاعه كُلّ فِي قَوْله : { لِكُلِّ أَوَّاب } وَالرَّفْع عَلَى الِاسْتِئْنَاف , وَهُوَ مُرَاد بِهِ الْجَزَاء مَنْ خَشِيَ الرَّحْمَن بِالْغَيْبِ , قِيلَ لَهُ اُدْخُلْ الْجَنَّة ; فَيَكُون حِينَئِذٍ قَوْله : { اُدْخُلُوهَا بِسَلَامٍ } جَوَابًا لِلْجَزَاءِ أَضْمَرَ قَبْله الْقَوْل , وَحَمَلَ فِعْلًا لِلْجَمِيعِ ; لِأَنَّ { مَنْ } قَدْ تَكُون فِي مَذْهَب الْجَمِيع .

وَجَاءَ بِقَلْبٍ مُنِيبٍ

وَقَوْله : { وَجَاءَ بِقَلْبٍ مُنِيب } يَقُول : وَجَاءَ اللَّه بِقَلْبٍ تَائِب مِنْ ذُنُوبه , رَاجِع مِمَّا يَكْرَههُ اللَّه إِلَى مَا يُرْضِيه . كَمَا : 24743 - حَدَّثَنَا بِشْر , قَالَ : ثَنَا يَزِيد , قَالَ : ثَنَا سَعِيد , عَنْ قَتَادَة , قَوْله : { وَجَاءَ بِقَلْبٍ مُنِيب } : أَيْ مُنِيب إِلَى رَبّه مُقْبِل .
