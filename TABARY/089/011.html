الَّذِينَ طَغَوْا فِي الْبِلَادِ

وَقَوْله : { الَّذِينَ طَغَوْا فِي الْبِلَاد } يَعْنِي بِقَوْلِهِ جَلَّ ثَنَاؤُهُ { الَّذِينَ } عَادًا وَثَمُود وَفِرْعَوْن وَجُنْده . وَيَعْنِي بِقَوْلِهِ : { طَغَوْا } : تَجَاوَزُوا مَا أَبَاحَهُ لَهُمْ رَبّهمْ , وَعَتَوْا عَلَى رَبّهمْ إِلَى مَا حَظَرَهُ عَلَيْهِمْ مِنْ الْكُفْر بِهِ وَقَوْله { فِي الْبِلَاد } : الَّتِي كَانُوا فِيهَا .
