وَالَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ لَنُدْخِلَنَّهُمْ فِي الصَّالِحِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَاَلَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَات لَأُدْخِلَنهُمْ فِي الصَّالِحِينَ } يَقُول تَعَالَى ذِكْره : { وَاَلَّذِينَ آمَنُوا } بِاَللَّهِ وَرَسُوله { وَعَمِلُوا الصَّالِحَات } مِنْ الْأَعْمَال , وَذَلِكَ أَنْ يُؤَدُّوا فَرَائِض اللَّه , وَيَجْتَنِبُوا مَحَارِمه { لَأُدْخِلَنهُمْ فِي الصَّالِحِينَ } فِي مَدْخَل الصَّالِحِينَ , وَذَلِكَ الْجَنَّة .
