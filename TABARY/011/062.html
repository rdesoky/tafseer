قَالُوا يَا صَالِحُ قَدْ كُنْتَ فِينَا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالُوا يَا صَالِح قَدْ كُنْت فِينَا مَرْجُوًّا } يَقُول تَعَالَى ذِكْره : قَالَتْ ثَمُود لِصَالِحٍ نَبِيّهمْ : { يَا صَالِح قَدْ كُنْت فِينَا مَرْجُوًّا } أَيْ كُنَّا نَرْجُو أَنْ تَكُون فِينَا سَيِّدًا

مَرْجُوًّا قَبْلَ

{ قَبْل هَذَا } الْقَوْل الَّذِي قُلْته لَنَا مِنْ أَنَّهُ مَا لَنَا مِنْ إِلَه غَيْر اللَّه .

هَذَا أَتَنْهَانَا أَنْ نَعْبُدَ مَا يَعْبُدُ

{ أَتَنْهَانَا أَنْ نَعْبُد مَا يَعْبُد آبَاؤُنَا } يَقُول : أَتَنْهَانَا أَنْ نَعْبُد الْآلِهَة الَّتِي كَانَتْ آبَاؤُنَا تَعْبُد

آبَاؤُنَا وَإِنَّنَا لَفِي شَكٍّ مِمَّا تَدْعُونَا إِلَيْهِ

{ وَإِنَّنَا لَفِي شَكّ مِمَّا تَدْعُونَا إِلَيْهِ مُرِيب } يَعْنُونَ أَنَّهُمْ لَا يَعْلَمُونَ صِحَّة مَا يَدْعُوهُمْ إِلَيْهِ مِنْ تَوْحِيد اللَّه , وَأَنَّ الْأُلُوهَة لَا تَكُون إِلَّا لَهُ خَالِصًا . وَقَوْله { مُرِيب } أَيْ يُوجِب التُّهْمَة مِنْ أَرَبْته فَأَنَا أَرِيبه إِرَابَة , إِذَا فَعَلْت بِهِ فِعْلًا يُوجِب لَهُ الرِّيبَة , وَمِنْهُ قَوْل الْهُذَلِيّ : كُنْت إِذَا أَتَوْته مِنْ غَيْب يَشُمّ عِطْفِي وَيُبَزّ ثَوْبِي كَأَنَّمَا أَرَبْته بِرَيْبِ
