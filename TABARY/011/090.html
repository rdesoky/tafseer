وَاسْتَغْفِرُوا رَبَّكُمْ ثُمَّ تُوبُوا إِلَيْهِ إِنَّ رَبِّي رَحِيمٌ وَدُودٌ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَاسْتَغْفِرُوا رَبّكُمْ ثُمَّ تُوبُوا إِلَيْهِ إِنَّ رَبِّي رَحِيم وَدُود } يَقُول تَعَالَى ذِكْره مُخْبِرًا عَنْ قِيل شُعَيْب لِقَوْمِهِ : { اِسْتَغْفِرُوا رَبّكُمْ } أَيّهَا الْقَوْم مِنْ ذُنُوبكُمْ بَيْنكُمْ وَبَيْن رَبّكُمْ الَّتِي أَنْتُمْ عَلَيْهَا مُقِيمُونَ مِنْ عِبَادَة الْآلِهَة وَالْأَصْنَام وَبَخْس النَّاس حُقُوقهمْ فِي الْمَكَايِيل وَالْمَوَازِين . { ثُمَّ تُوبُوا إِلَيْهِ } يَقُول : ثُمَّ اِرْجِعُوا إِلَى طَاعَته وَالِانْتِهَاء إِلَى أَمْره وَنَهْيه . { إِنَّ رَبِّي رَحِيم } يَقُول : هُوَ رَحِيم بِمَنْ تَابَ وَأَنَابَ إِلَيْهِ أَنْ يُعَذِّبهُ بَعْد التَّوْبَة . { وَدُود } يَقُول : ذُو مَحَبَّة لِمَنْ أَنَابَ وَتَابَ إِلَيْهِ يَوَدّهُ وَيُحِبّهُ .
