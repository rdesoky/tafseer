وَيَا قَوْمِ هَذِهِ نَاقَةُ اللَّهِ لَكُمْ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَيَا قَوْم هَذِهِ نَاقَة اللَّه لَكُمْ آيَة } يَقُول تَعَالَى ذِكْره مُخْبِرًا عَنْ قِيل صَالِح لِقَوْمِهِ مِنْ ثَمُود إِذْ قَالُوا لَهُ { وَإِنَّنَا لَفِي شَكّ مِمَّا تَدْعُونَا إِلَيْهِ مُرِيب } وَسَأَلُوهُ الْآيَة عَلَى مَا دَعَاهُمْ إِلَيْهِ : { يَا قَوْم هَذِهِ نَاقَة اللَّه لَكُمْ آيَة } يَقُول : حُجَّة وَعَلَامَة , وَدَلَالَة عَلَى حَقِيقَة مَا أَدْعُوكُمْ إِلَيْهِ .

آيَةً فَذَرُوهَا تَأْكُلْ فِي أَرْضِ

{ فَذَرُوهَا تَأْكُل فِي أَرْض اللَّه } فَلَيْسَ عَلَيْكُمْ رِزْقهَا وَلَا مُؤْنَتهَا .

اللَّهِ وَلَا تَمَسُّوهَا

{ وَلَا تَمَسُّوهَا بِسُوءٍ } يَقُول : لَا تَقْتُلُوهَا وَلَا تَنَالُوهَا بِعُقْرٍ ,

بِسُوءٍ فَيَأْخُذَكُمْ عَذَابٌ

{ فَيَأْخُذكُمْ عَذَاب قَرِيب } يَقُول : فَإِنَّكُمْ إِنْ تَمَسُّوهَا بِسُوءٍ يَأْخُذكُمْ عَذَاب مِنْ اللَّه غَيْر بَعِيد فَيُهْلِككُمْ .
