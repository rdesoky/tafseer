فَفَهَّمْنَاهَا سُلَيْمَانَ

وَقَوْله : { فَفَهَّمْنَاهَا } يَقُول : فَفَهَّمْنَا الْقَضِيَّة فِي ذَلِكَ { سُلَيْمَان } دُون دَاوُد .

وَكُلًّا آتَيْنَا حُكْمًا وَعِلْمًا

يَقُول : وَكُلّهمْ مِنْ دَاوُد وَسُلَيْمَان وَالرُّسُل الَّذِينَ ذَكَرَهُمْ فِي أَوَّل هَذِهِ السُّورَة آتَيْنَا حُكْمًا وَهُوَ النُّبُوَّة , وَعِلْمًا : يَعْنِي وَعِلْمًا بِأَحْكَامِ اللَّه .

وَسَخَّرْنَا مَعَ دَاوُدَ الْجِبَالَ يُسَبِّحْنَ وَالطَّيْرَ

وَقَوْله : { وَسَخَّرْنَا مَعَ دَاوُد الْجِبَال يُسَبِّحْنَ وَالطَّيْر } يَقُول تَعَالَى ذِكْره : وَسَخَّرْنَا مَعَ دَاوُد الْجِبَال وَالطَّيْر يُسَبِّحْنَ مَعَهُ إِذَا سَبَّحَ . وَكَانَ قَتَادَة يَقُول فِي مَعْنَى قَوْله : { يُسَبِّحْنَ } فِي هَذَا الْمَوْضِع مَا : 18668 - حَدَّثَنَا بِهِ بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثَنَا سَعِيد , عَنْ قَتَادَة , قَوْله : { وَسَخَّرْنَا مَعَ دَاوُد الْجِبَال يُسَبِّحْنَ وَالطَّيْر } : أَيْ يُصَلِّينَ مَعَ دَاوُد إِذَا صَلَّى .

وَكُنَّا فَاعِلِينَ

وَقَوْله : { وَكُنَّا فَاعِلِينَ } يَقُول : وَكُنَّا قَدْ قَضَيْنَا أَنَّا فَاعِلُو ذَلِكَ , وَمُسَخِّرُو الْجِبَال وَالطَّيْر فِي أُمّ الْكِتَاب مَعَ دَاوُد عَلَيْهِ الصَّلَاة وَالسَّلَام .
