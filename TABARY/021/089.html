وَزَكَرِيَّا إِذْ نَادَى رَبَّهُ رَبِّ لَا تَذَرْنِي فَرْدًا وَأَنْتَ خَيْرُ الْوَارِثِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَزَكَرِيَّا إِذَا نَادَى رَبّه رَبّ لَا تَذَرنِي فَرْدًا وَأَنْتَ خَيْر الْوَارِثِينَ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : وَاذْكُرْ يَا مُحَمَّد زَكَرِيَّا حِين نَادَى رَبّه { رَبّ لَا تَذَرنِي } وَحِيدًا { فَرْدًا } لَا وَلَد لِي وَلَا عَقِب { وَأَنْتَ خَيْر الْوَارِثِينَ } يَقُول : فَارْزُقْنِي وَارِثًا مِنْ آل يَعْقُوب يَرِثنِي . ثُمَّ رَدَّ الْأَمْر إِلَى اللَّه فَقَالَ : { وَأَنْتَ خَيْر الْوَارِثِينَ } .
