قَالَ لَقَدْ كُنْتُمْ أَنْتُمْ وَآبَاؤُكُمْ فِي ضَلَالٍ مُبِينٍ

{ قَالَ } إِبْرَاهِيم : { لَقَدْ كُنْتُمْ } أَيّهَا الْقَوْم { أَنْتُمْ وَآبَاؤُكُمْ } بِعِبَادَتِكُمْ إِيَّاهَا { فِي ضَلَال مُبِين } يَقُول : فِي ذَهَاب عَنْ سَبِيل الْحَقّ , وَجَوْر عَنْ قَصْد السَّبِيل مُبِين ; يَقُول : بَيِّن لِمَنْ تَأَمَّلَهُ بِعَقْلٍ أَنَّكُمْ كَذَلِكَ فِي جَوْر عَنْ الْحَقّ .
