وَإِنَّ لَكُمْ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَإِنَّ لَكُمْ } . يَقُول تَعَالَى ذِكْره : { وَإِنَّ لَكُمْ } أَيّهَا النَّاس

فِي الْأَنْعَامِ لَعِبْرَةً

{ فِي الْأَنْعَام لَعِبْرَة } تَعْتَبِرُونَ بِهَا , فَتَعْرِفُونَ بِهَا أَيَادِي اللَّه عِنْدكُمْ وَقُدْرَته عَلَى مَا يَشَاء , وَأَنَّهُ الَّذِي لَا يَمْتَنِع عَلَيْهِ شَيْء أَرَادَهُ وَلَا يُعْجِزهُ شَيْء شَاءَهُ .

نُسْقِيكُمْ مِمَّا فِي بُطُونِهَا

{ نُسْقِيكُمْ مِمَّا فِي بُطُونهَا } مِنَ اللَّبَن الْخَارِج مِنْ بَيْن الْفَرْث وَالدَّم.

وَلَكُمْ

{ وَلَكُمْ } مَعَ ذَلِكَ

فِيهَا

{ فِيهَا } يَعْنِي فِي الْأَنْعَام ,

مَنَافِعُ كَثِيرَةٌ

{ مَنَافِع كَثِيرَة } وَذَلِكَ كَالْإِبِلِ الَّتِي يُحْمَل عَلَيْهَا وَيُرْكَب ظَهْرهَا وَيُشْرَب دَرّهَا .

وَمِنْهَا تَأْكُلُونَ

{ وَمِنْهَا تَأْكُلُونَ } يَعْنِي مِنْ لُحُومهَا تَأْكُلُونَ .
