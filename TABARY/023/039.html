قَالَ رَبِّ انْصُرْنِي بِمَا كَذَّبُونِ

وَقَوْله : { قَالَ رَبِّ انْصُرْنِي بِمَا كَذَّبُونِ } يَقُول : قَالَ صَالِح لَمَّا أَيِسَ مِنْ إِيمَان قَوْمه بِاللَّهِ وَمِنْ تَصْدِيقهمْ إِيَّاهُ بِقَوْلِهِمْ { وَمَا نَحْنُ لَهُ بِمُؤْمِنِينَ } { رَبِّ انْصُرْنِي } عَلَى هَؤُلَاءِ { بِمَا كَذَّبُونِ } يَقُول : بِتَكْذِيبِهِمْ إِيَّايَ فِيمَا دَعَوْتهمْ إِلَيْهِ مِنَ الْحَقّ , فَاسْتَغَاثَ صَلَوَات اللَّه عَلَيْهِ بِرَبِّهِ مِنْ أَذَاهُمْ إِيَّاهُ وَتَكْذِيبهمْ لَهُ ,
