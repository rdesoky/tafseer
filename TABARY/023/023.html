وَلَقَدْ أَرْسَلْنَا نُوحًا إِلَى قَوْمِهِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَقَدْ أَرْسَلْنَا نُوحًا إِلَى قَوْمه } يَقُول تَعَالَى ذِكْره : { وَلَقَدْ أَرْسَلْنَا نُوحًا إِلَى قَوْمه } دَاعِيهمْ إِلَى طَاعَتنَا وَتَوْحِيدنَا وَالْبَرَاءَة مِنْ كُلّ مَعْبُود سِوَانَا .

فَقَالَ يَا قَوْمِ اعْبُدُوا

{ فَقَالَ } لَهُمْ نُوح : { يَا قَوْمِ اعْبُدُوا اللَّه } يَقُول : قَالَ لَهُمْ : ذِلُّوا يَا قَوْم لِلَّهِ بِالطَّاعَةِ.

اللَّهَ مَا لَكُمْ مِنْ إِلَهٍ

{ مَا لَكُمْ مِنْ إِلَه غَيْره } يَقُول : مَا لَكُمْ مِنْ مَعْبُود يَجُوز لَكُمْ أَنْ تَعْبُدُوهُ غَيْره .

غَيْرُهُ أَفَلَا

{ أَفَلَا تَتَّقُونَ } يَقُول : أَفَلَا تَخْشَوْنَ بِعِبَادَتِكُمْ غَيْره عِقَابه أَنْ يَحِلّ بِكُمْ .
