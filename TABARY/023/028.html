فَإِذَا اسْتَوَيْتَ أَنْتَ وَمَنْ مَعَكَ عَلَى الْفُلْكِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَإِذَا اسْتَوَيْت أَنْتَ وَمَنْ مَعَك عَلَى الْفُلْك } يَعْنِي تَعَالَى ذِكْره بِقَوْلِهِ : { فَإِذَا اسْتَوَيْت أَنْتَ وَمَنْ مَعَك عَلَى الْفُلْك } فَإِذَا اعْتَدَلْت فِي السَّفِينَة أَنْتَ وَمَعَك مِمَّنْ حَمَلْته مَعَك مِنْ أَهْلك , رَاكِبًا فِيهَا عَالِيًا فَوْقهَا ;

فَقُلِ الْحَمْدُ لِلَّهِ الَّذِي نَجَّانَا مِنَ الْقَوْمِ الظَّالِمِينَ

{ فَقُلِ الْحَمْد لِلَّهِ الَّذِي نَجَّانَا مِنَ الْقَوْم الظَّالِمِينَ } يَعْنِي مِنَ الْمُشْرِكِينَ .
