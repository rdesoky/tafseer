قَالَ رَبِّ انْصُرْنِي بِمَا كَذَّبُونِ

وَقَوْله : { قَالَ رَبِّ انْصُرْنِي بِمَا كَذَّبُونِ } يَقُول : قَالَ نُوح دَاعِيًا رَبّه مُسْتَنْصِرًا بِهِ عَلَى قَوْمه , لَمَّا طَالَ أَمْره وَأَمْرهُمْ وَتَمَادَوْا فِي غَيّهمْ : { رَبِّ انْصُرْنِي } عَلَى قَوْمِي { بِمَا كَذَّبُونِ } يَعْنِي بِتَكْذِيبِهِمْ إِيَّايَ , فِيمَا بَلَّغْتهمْ مِنْ رِسَالَتك وَدَعَوْتهمْ إِلَيْهِ مِنْ تَوْحِيدك .
