وَالَّذِينَ هُمْ لِأَمَانَاتِهِمْ وَعَهْدِهِمْ رَاعُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَالَّذِينَ هُمْ لِأَمَانَاتِهِمْ وَعَهْدهمْ رَاعُونَ } يَقُول تَعَالَى ذِكْره : { وَالَّذِينَ هُمْ لِأَمَانَاتِهِمْ } الَّتِي اؤْتُمِنُوا عَلَيْهَا وَعَهْدهمْ , وَهُوَ عُقُودهمْ الَّتِي عَاقَدُوا النَّاس ; { رَاعُونَ } يَقُول : حَافِظُونَ لَا يُضَيِّعُونَ , وَلَكِنَّهُمْ يُوفُونَ بِذَلِكَ كُلّه . وَاخْتَلَفَتِ الْقُرَّاء فِي قِرَاءَة ذَلِكَ , فَقَرَأَتْهُ عَامَّة قُرَّاء الْأَمْصَار إِلَّا ابْن كَثِير : { وَالَّذِينَ هُمْ لِأَمَانَاتِهِمْ } عَلَى الْجَمْع , وَقَرَأَ ذَلِكَ ابْن كَثِير : " لِأَمَانَتِهِمْ " عَلَى الْوَاحِدَة . وَالصَّوَاب مِنَ الْقِرَاءَة فِي ذَلِكَ عِنْدنَا : { لِأَمَانَاتِهِمْ } لِإِجْمَاعِ الْحُجَّة مِنَ الْقُرَّاء عَلَيْهَا .
