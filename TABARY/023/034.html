وَلَئِنْ أَطَعْتُمْ بَشَرًا مِثْلَكُمْ إِنَّكُمْ إِذًا لَخَاسِرُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَئِنْ أَطَعْتُمْ بَشَرًا مِثْلكُمْ إِنَّكُمْ إِذًا لَخَاسِرُونَ } يَقُول تَعَالَى ذِكْره مُخْبِرًا عَنْ قِيل الْمَلَأ مِنْ قَوْم صَالِح لِقَوْمِهِمْ : { وَلَئِنْ أَطَعْتُمْ بَشَرًا مِثْلكُمْ } فَاتَّبَعْتُمُوهُ وَقَبِلْتُمْ مَا يَقُول وَصَدَّقْتُمُوهُ . { إِنَّكُمْ } أَيّهَا الْقَوْم { إِذًا لَخَاسِرُونَ } يَقُول : قَالُوا : إِنَّكُمْ إِذَنْ لَمَغْبُونُونَ حُظُوظكُمْ مِنَ الشَّرَف وَالرِّفْعَة فِي الدُّنْيَا , بِاتِّبَاعِكُمْ إِيَّاهُ .
