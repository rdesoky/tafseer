وَأَقِيمُوا الصَّلَاةَ وَآتُوا الزَّكَاةَ وَأَطِيعُوا الرَّسُولَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَأَقِيمُوا الصَّلَاة وَآتُوا الزَّكَاة وَأَطِيعُوا الرَّسُول } يَقُول تَعَالَى ذِكْره : { وَأَقِيمُوا } أَيّهَا النَّاس { الصَّلَاة } بِحُدُودِهَا فَلَا تُضَيِّعُوهَا . { وَآتُوا الزَّكَاة } الَّتِي فَرَضَهَا اللَّه عَلَيْكُمْ أَهْلهَا , وَأَطِيعُوا رَسُول رَبّكُمْ فِيمَا أَمَرَكُمْ وَنَهَاكُمْ.

لَعَلَّكُمْ تُرْحَمُونَ

يَقُول : كَيْ يَرْحَمكُمْ رَبّكُمْ فَيُنَجِّيكُمْ مِنْ عَذَابه ,
