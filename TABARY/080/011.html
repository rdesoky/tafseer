كَلَّا إِنَّهَا تَذْكِرَةٌ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { كَلَّا إِنَّهَا تَذْكِرَة } يَقُول تَعَالَى ذِكْره : { كَلَّا } مَا الْأَمْر كَمَا تَفْعَل يَا مُحَمَّد , مِنْ أَنْ تَعْبِس فِي وَجْه مَنْ جَاءَك يَسْعَى وَهُوَ يَخْشَى , وَتَتَصَدَّى لِمَنْ اِسْتَغْنَى { إِنَّهَا تَذْكِرَة } يَقُول : إِنَّ هَذِهِ الْعِظَة وَهَذِهِ السُّورَة { تَذْكِرَة } يَقُول : عِظَة وَعِبْرَة .
