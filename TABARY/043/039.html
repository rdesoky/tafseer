وَلَنْ يَنْفَعَكُمُ الْيَوْمَ إِذْ ظَلَمْتُمْ أَنَّكُمْ فِي الْعَذَابِ مُشْتَرِكُونَ

وَقَوْله : { وَلَنْ يَنْفَعكُمُ الْيَوْم } أَيّهَا الْعَاشُونَ عَنْ ذِكْر اللَّه فِي الدُّنْيَا { إِذْ ظَلَمْتُمْ أَنَّكُمْ فِي الْعَذَاب مُشْتَرِكُونَ } يَقُول : لَنْ يُخَفِّف عَنْكُمُ الْيَوْم مِنْ عَذَاب اللَّه اشْتِرَاككُمْ فِيهِ ; لِأَنَّ لِكُلِّ وَاحِد مِنْكُمْ نَصِيبه مِنْهُ , و " أَنَّ " مِنْ قَوْله { أَنَّكُمْ } فِي مَوْضِع رَفْع لِمَا ذَكَرْت أَنَّ مَعْنَاهُ : لَنْ يَنْفَعكُمْ اشْتِرَاككُمْ .
