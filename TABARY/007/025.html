قَالَ فِيهَا تَحْيَوْنَ وَفِيهَا تَمُوتُونَ وَمِنْهَا تُخْرَجُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ فِيهَا تَحْيَوْنَ وَفِيهَا تَمُوتُونَ وَمِنْهَا تُخْرَجُونَ } يَقُول تَعَالَى ذِكْره : قَالَ اللَّه لِلَّذِينَ أَهْبَطَهُمْ مِنْ سَمَاوَاتِهِ إِلَى أَرْضه : { فِيهَا تَحْيَوْنَ } يَقُول : فِي الْأَرْض تَحْيَوْنَ , يَقُول : تَكُونُونَ فِيهَا أَيَّام حَيَاتكُمْ , { وَفِيهَا تَمُوتُونَ } يَقُول فِي الْأَرْض تَكُون وَفَاتكُمْ , { وَمِنْهَا تُخْرَجُونَ } : يَقُول : وَمِنْ الْأَرْض يُخْرِجكُمْ رَبّكُمْ , وَيَحْشُركُمْ إِلَيْهِ لِبَعْثِ الْقِيَامَة أَحْيَاء .
