قَالَ يَا مُوسَى إِنِّي اصْطَفَيْتُكَ عَلَى النَّاسِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ يَا مُوسَى إِنِّي اِصْطَفَيْتُك عَلَى النَّاس بِرِسَالَاتِي } يَقُول تَعَالَى ذِكْره : قَالَ اللَّه لِمُوسَى : { يَا مُوسَى إِنِّي اِصْطَفَيْتُك عَلَى النَّاس } يَقُول : اِخْتَرْتُك عَلَى النَّاس { بِرِسَالَاتِي } إِلَى خَلْقِي , أَرْسَلْتُك بِهَا إِلَيْهِمْ .

بِرِسَالَاتِي

{ وَبِكَلَامِي } كَلَّمْتُك وَنَاجَيْتُك دُون غَيْرك مِنْ خَلْقِي .

وَبِكَلَامِي فَخُذْ مَا آتَيْتُكَ وَكُنْ مِنَ

{ فَخُذْ مَا آتَيْتُك } يَقُول : فَخُذْ مَا أَعْطَيْتُك مِنْ أَمْرِي وَنَهْيِي وَتَمَسَّكْ بِهِ , وَاعْمَلْ بِهِ , يُرِيد { وَكُنْ مِنْ الشَّاكِرِينَ } لِلَّهِ عَلَى مَا آتَاك مِنْ رِسَالَته , وَحَصِّلْ بِهِ مِنْ النَّجْوَى بِطَاعَتِهِ فِي أَمْره وَنَهْيه وَالْمُسَارَعَة إِلَى رِضَاهُ.
