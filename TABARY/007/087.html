وَإِنْ كَانَ طَائِفَةٌ مِنْكُمْ آمَنُوا بِالَّذِي أُرْسِلْتُ بِهِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَإِنْ كَانَ طَائِفَة مِنْكُمْ آمَنُوا بِاَلَّذِي أُرْسِلْت بِهِ } يَعْنِي بِقَوْلِهِ تَعَالَى ذِكْره : { وَإِنْ كَانَ طَائِفَة مِنْكُمْ } وَإِنْ كَانَتْ جَمَاعَة مِنْكُمْ وَفِرْقَة آمَنُوا , يَقُول : صَدَّقُوا , { بِاَلَّذِي أُرْسِلْت بِهِ } مِنْ إِخْلَاص الْعِبَادَة لِلَّهِ وَتَرْك مَعَاصِيه وَظُلْم النَّاس وَبَخْسهمْ فِي الْمَكَايِيل وَالْمَوَازِين , فَاتَّبَعُونِي عَلَى ذَلِكَ .

وَطَائِفَةٌ لَمْ يُؤْمِنُوا

{ وَطَائِفَة لَمْ يُؤْمِنُوا } يَقُول : وَجَمَاعَة أُخْرَى لَمْ يُصَدِّقُوا بِذَلِكَ , وَلَمْ يَتَّبِعُونِي عَلَيْهِ .

فَاصْبِرُوا حَتَّى يَحْكُمَ اللَّهُ بَيْنَنَا

{ فَاصْبِرُوا حَتَّى يَحْكُم اللَّه بَيْننَا } يَقُول : فَاحْتَبِسُوا عَلَى قَضَاء اللَّه الْفَاصِل بَيْننَا وَبَيْنكُمْ .

وَهُوَ خَيْرُ الْحَاكِمِينَ

{ وَهُوَ خَيْر الْحَاكِمِينَ } يَقُول : وَاَللَّه خَيْر مَنْ يَفْصِل وَأَعْدَل مَنْ يَقْضِي , لِأَنَّهُ لَا يَقَع فِي حُكْمه مَيْل إِلَى أَحَد , وَلَا مُحَابَاة لِأَحَدٍ ; وَاَللَّه أَعْلَم.
