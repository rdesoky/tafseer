قَالَ الَّذِينَ اسْتَكْبَرُوا إِنَّا بِالَّذِي آمَنْتُمْ بِهِ كَافِرُونَ

قَالَ الَّذِينَ اِسْتَكْبَرُوا عَنْ أَمْر اللَّه وَأَمْر رَسُوله صَالِح : { إِنَّا } أَيّهَا الْقَوْم { بِاَلَّذِي آمَنْتُمْ بِهِ } يَقُول : صَدَّقْتُمْ بِهِ مِنْ نُبُوَّة صَالِح , وَأَنَّ الَّذِي جَاءَ بِهِ حَقّ مِنْ عِنْد اللَّه { كَافِرُونَ } يَقُول جَاحِدُونَ مُنْكِرُونَ , لَا نُصَدِّق بِهِ وَلَا نُقِرّ .
