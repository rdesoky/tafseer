قَالَ الْمَلَأُ الَّذِينَ كَفَرُوا مِنْ قَوْمِهِ إِنَّا لَنَرَاكَ فِي سَفَاهَةٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ الْمَلَأ الَّذِينَ كَفَرُوا مِنْ قَوْمه إِنَّا لَنَرَاك فِي سَفَاهَة } يَقُول تَعَالَى ذِكْره مُخْبِرًا عَمَّا أَجَابَ هُودًا بِهِ قَوْمه الَّذِينَ كَفَرُوا بِاَللَّهِ : { قَالَ الْمَلَأ الَّذِينَ كَفَرُوا } يَعْنِي الَّذِينَ جَحَدُوا تَوْحِيد اللَّه , وَأَنْكَرُوا رِسَالَة هُود إِلَيْهِمْ : { إِنَّا لَنَرَاك } يَا هُود { فِي سَفَاهَة } يَعْنُونَ فِي ضَلَالَة عَنْ الْحَقّ وَالصَّوَاب , بِتَرْكِك دِيننَا وَعِبَادَة آلِهَتنَا .

وَإِنَّا لَنَظُنُّكَ مِنَ الْكَاذِبِينَ

{ وَإِنَّا لَنَظُنّك مِنْ الْكَاذِبِينَ } فِي قِيلك إِنِّي رَسُول مِنْ رَبّ الْعَالَمِينَ .
