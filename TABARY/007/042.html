وَالَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ لَا نُكَلِّفُ نَفْسًا إِلَّا وُسْعَهَا أُولَئِكَ أَصْحَابُ الْجَنَّةِ هُمْ فِيهَا خَالِدُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَاَلَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَات لَا نُكَلِّف نَفْسًا إِلَّا وُسْعهَا أُولَئِكَ أَصْحَاب الْجَنَّة هُمْ فِيهَا خَالِدُونَ } يَقُول جَلَّ ثَنَاؤُهُ : وَاَلَّذِينَ صَدَقُوا اللَّه وَرَسُوله وَأَقَرُّوا بِمَا جَاءَهُمْ بِهِ مِنْ وَحْي اللَّه وَتَنْزِيله وَشَرَائِع دِينه , وَعَمِلُوا مَا أَمَرَهُمْ اللَّه بِهِ فَأَطَاعُوهُ وَتَجَنَّبُوا مَا نَهَاهُمْ عَنْهُ. { لَا نُكَلِّف نَفْسًا إِلَّا وُسْعهَا } يَقُول : لَا نُكَلِّف نَفْسًا مِنْ الْأَعْمَال إِلَّا مَا يَسَعهَا فَلَا تَحْرَج فِيهِ { أُولَئِكَ } يَقُول : هَؤُلَاءِ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَات , { أَصْحَاب الْجَنَّة } يَقُول : هُمْ أَهْل الْجَنَّة الَّذِينَ هُمْ أَهْلهَا دُون غَيْرهمْ مِمَّنْ كَفَرَ بِاَللَّهِ , وَعَمِلَ بِسَيِّئَاتِهِمْ { فِيهَا خَالِدُونَ } يَقُول : هُمْ فِي الْجَنَّة مَاكِثُونَ , دَائِم فِيهَا مُكْثهمْ لَا يَخْرُجُونَ مِنْهَا وَلَا يُسْلَبُونَ نَعِيمهمْ .
