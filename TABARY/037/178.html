وَتَوَلَّ عَنْهُمْ حَتَّى حِينٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَتَوَلَّ عَنْهُمْ حَتَّى حِين } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : وَأَعْرِضْ يَا مُحَمَّد عَنْ هَؤُلَاءِ الْمُشْرِكِينَ , وَخَلّهمْ وَقَرْيَتهمْ عَلَى رَبّهمْ { حَتَّى حِين } يَقُول : إِلَى حِين يَأْذَن اللَّه بِهَلَاكِهِمْ
