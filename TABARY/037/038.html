إِنَّكُمْ لَذَائِقُو الْعَذَابِ الْأَلِيمِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّكُمْ لَذَائِقُوا الْعَذَاب الْأَلِيم } يَقُول تَعَالَى ذِكْره لِهَؤُلَاءِ الْمُشْرِكِينَ مِنْ أَهْل مَكَّة , الْقَائِلِينَ لِمُحَمَّدٍ : شَاعِر مَجْنُون { إِنَّكُمْ } أَيّهَا الْمُشْرِكُونَ { لَذَائِقُوا الْعَذَاب الْأَلِيم } الْمُوجِع فِي الْآخِرَة
