وَبَارَكْنَا عَلَيْهِ وَعَلَى إِسْحَاقَ

وَقَوْله : { وَبَارَكْنَا عَلَيْهِ وَعَلَى إِسْحَاق } يَقُول تَعَالَى ذِكْره : وَبَارَكْنَا عَلَى إِبْرَاهِيم وَعَلَى إِسْحَاق

وَمِنْ ذُرِّيَّتِهِمَا مُحْسِنٌ

{ وَمِنْ ذُرِّيَّتهمَا مُحْسِن } يَعْنِي بِالْمُحْسِنِ : الْمُؤْمِن الْمُطِيع لِلَّهِ , الْمُحْسِن فِي طَاعَته إِيَّاهُ

وَظَالِمٌ لِنَفْسِهِ مُبِينٌ

{ وَظَالِم لِنَفْسِهِ مُبِين } وَيَعْنِي بِالظَّالِمِ لِنَفْسِهِ : الْكَافِر بِاَللَّهِ , الْجَالِب عَلَى نَفْسه بِكُفْرِهِ عَذَاب اللَّه وَأَلِيم عِقَابه { مُبِين } : يَعْنِي الَّذِي قَدْ أَبَانَ ظُلْمه نَفْسه بِكُفْرِهِ بِاَللَّهِ . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 22674- حَدَّثَنَا مُحَمَّد بْن الْحُسَيْن , قَالَ : ثنا أَحْمَد بْن الْمُفَضَّل , قَالَ : ثنا أَسْبَاط , عَنْ السُّدِّيّ , فِي قَوْله : { مُحْسِن وَظَالِم لِنَفْسِهِ مُبِين } قَالَ : الْمُحْسِن : الْمُطِيع لِلَّهِ , وَالظَّالِم لِنَفْسِهِ : الْعَاصِي لِلَّهِ .
