وَنَادَيْنَاهُ أَنْ يَا

قَوْله : { وَنَادَيْنَاهُ أَنْ يَا إِبْرَاهِيم قَدْ صَدَّقْت الرُّؤْيَا } وَهَذَا جَوَاب قَوْله : { فَلَمَّا أَسْلَمَا } وَمَعْنَى الْكَلَام : فَلَمَّا أَسْلَمَا وَتَلَّهُ لِلْجَبِينِ , وَنَادَيْنَاهُ أَنْ يَا إِبْرَاهِيم ; وَأُدْخِلَتْ الْوَاو فِي ذَلِكَ كَمَا أُدْخِلَتْ فِي قَوْله : { حَتَّى إِذَا جَاءُوهَا وَفُتِحَتْ أَبْوَابهَا } 39 73 وَقَدْ تَفْعَل الْعَرَب ذَلِكَ فَتُدْخِل الْوَاو فِي جَوَاب فَلَمَّا , وَحَتَّى وَإِذَا تُلْقِيهَا.
