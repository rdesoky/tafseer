قَالُوا لَئِنْ لَمْ تَنْتَهِ يَا لُوطُ لَتَكُونَنَّ مِنَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالُوا لَئِنْ لَمْ تَنْتَهِ يَا لُوط لَتَكُونَن مِنْ الْمُخْرَجِينَ } يَقُول تَعَالَى ذِكْره : قَالَ قَوْم لُوط : { لَئِنْ لَمْ تَنْتَهِ يَا لُوط } عَنْ نَهْينَا عَنْ إِتْيَان الذُّكْرَان { لَتَكُونَن مِنْ الْمُخْرَجِينَ } مِنْ بَيْن أَظْهُرنَا وَبَلَدنَا .
