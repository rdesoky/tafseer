وَمَا أَهْلَكْنَا مِنْ قَرْيَةٍ إِلَّا لَهَا مُنْذِرُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَمَا أَهْلَكْنَا مِنْ قَرْيَة إِلَّا لَهَا مُنْذِرُونَ } يَقُول تَعَالَى ذِكْره : { وَمَا أَهْلَكْنَا مِنْ قَرْيَة } مِنْ هَذِهِ الْقُرَى الَّتِي وُصِفَتْ فِي هَذِهِ السُّوَر { إِلَّا لَهَا مُنْذِرُونَ } يَقُول : إِلَّا بَعْد إِرْسَالنَا إِلَيْهِمْ رُسُلًا يُنْذِرُونَهُمْ بَأْسنَا عَلَى كُفْرهمْ وَسُخْطنَا عَلَيْهِمْ .
