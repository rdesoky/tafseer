وَمَا أَسْأَلُكُمْ عَلَيْهِ مِنْ أَجْرٍ إِنْ أَجْرِيَ إِلَّا عَلَى رَبِّ الْعَالَمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَمَا أَسْأَلكُمْ عَلَيْهِ مِنْ أَجْر إِنْ أَجْرِي إِلَّا عَلَى رَبّ الْعَالَمِينَ } يَقُول : { وَمَا أَسْأَلكُمْ } عَلَى نُصْحِي لَكُمْ مِنْ جَزَاء وَثَوَاب , مَا جَزَائِي وَثَوَابِي عَلَى ذَلِكَ { إِلَّا عَلَى رَبّ الْعَالَمِينَ } .
