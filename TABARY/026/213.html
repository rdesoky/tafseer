فَلَا تَدْعُ مَعَ اللَّهِ إِلَهًا آخَرَ فَتَكُونَ مِنَ الْمُعَذَّبِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَلَا تَدْعُ مَعَ اللَّه إِلَهًا آخَر فَتَكُون مِنْ الْمُعَذَّبِينَ } . يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { فَلَا تَدْعُ } يَا مُحَمَّد { مَعَ اللَّه إِلَهًا آخَر } أَيْ لَا تَعْبُد مَعَهُ مَعْبُودًا غَيْره { فَتَكُون مِنْ الْمُعَذَّبِينَ } فَيَنْزِل بِك مِنْ الْعَذَاب مَا نَزَلَ بِهَؤُلَاءِ الَّذِينَ خَالَفُوا أَمْرنَا وَعَبَدُوا غَيْرنَا .
