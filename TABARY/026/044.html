فَأَلْقَوْا حِبَالَهُمْ وَعِصِيَّهُمْ وَقَالُوا بِعِزَّةِ فِرْعَوْنَ إِنَّا لَنَحْنُ الْغَالِبُونَ

{ فَأَلْقَوْا حِبَالهمْ وَعِصِيّهمْ } مِنْ أَيْدِيهمْ { وَقَالُوا بِعِزَّةِ فِرْعَوْن } يَقُول : أَقْسَمُوا بِقُوَّةِ فِرْعَوْن وَشِدَّة سُلْطَانه , وَمَنَعَة مَمْلَكَته { إِنَّا لَنَحْنُ الْغَالِبُونَ } مُوسَى .
