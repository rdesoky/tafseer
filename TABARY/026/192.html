وَإِنَّهُ لَتَنْزِيلُ رَبِّ الْعَالَمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَإِنَّهُ لَتَنْزِيل رَبّ الْعَالَمِينَ } يَقُول تَعَالَى ذِكْره : وَإِنَّ هَذَا الْقُرْآن { لَتَنْزِيل رَبّ الْعَالَمِينَ } وَالْهَاء فِي قَوْله { وَإِنَّهُ } كِنَايَة الذِّكْر الَّذِي فِي قَوْله : { وَمَا يَأْتِيهِمْ مِنْ ذِكْر مِنْ الرَّحْمَن } 26 5 . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 20343 - حَدَّثَنَا الْحَسَن , قَالَ : أَخْبَرَنَا عَبْد الرَّزَّاق , قَالَ : أَخْبَرَنَا مَعْمَر , عَنْ قَتَادَة , فِي قَوْله : { لَتَنْزِيل رَبّ الْعَالَمِينَ } قَالَ : هَذَا الْقُرْآن .
