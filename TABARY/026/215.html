وَاخْفِضْ جَنَاحَكَ لِمَنِ اتَّبَعَكَ مِنَ الْمُؤْمِنِينَ

وَقَوْله : { وَاخْفِضْ جَنَاحك } يَقُول : وَأَلِنْ جَانِبك وَكَلَامك { لِمَنْ اِتَّبَعَك مِنْ الْمُؤْمِنِينَ } كَمَا. 20381 - حَدَّثَنِي يُونُس , قَالَ : أَخْبَرَنَا اِبْن وَهْب , قَالَ : قَالَ اِبْن زَيْد , فِي قَوْله : { وَاخْفِضْ جَنَاحك لِمَنْ اِتَّبَعَك مِنْ الْمُؤْمِنِينَ } قَالَ : يَقُول : لِنْ لَهُمْ .
