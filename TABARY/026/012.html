قَالَ رَبِّ إِنِّي أَخَافُ أَنْ يُكَذِّبُونِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ رَبّ إِنِّي أَخَاف أَنْ يُكَذِّبُونِ } يَقُول تَعَالَى ذِكْره : { قَالَ } مُوسَى لِرَبِّهِ { رَبّ إِنِّي أَخَاف } مِنْ قَوْم فِرْعَوْن الَّذِينَ أَمَرْتنِي أَنْ آتِيَهُمْ { أَنْ يُكَذِّبُونِ } بِقِيلِي لَهُمْ : إِنَّك أَرْسَلْتنِي إِلَيْهِمْ .
