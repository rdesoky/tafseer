فَلَمَّا جَاءَ السَّحَرَةُ قَالُوا لِفِرْعَوْنَ أَئِنَّ لَنَا لَأَجْرًا إِنْ كُنَّا نَحْنُ الْغَالِبِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَلَمَّا جَاءَ السَّحَرَة قَالُوا لِفِرْعَوْن أَئِنّ لَنَا لَأَجْرًا إِنْ كُنَّا نَحْنُ الْغَالِبِينَ } يَقُول تَعَالَى ذِكْره : { فَلَمَّا جَاءَ السَّحَرَة } فِرْعَوْن لِوَعْدِ مُوسَى وَمَوْعِد فِرْعَوْن { قَالُوا لِفِرْعَوْن أَئِنّ لَنَا لَأَجْرًا } سِحْرنَا قِبَلك { إِنْ كُنَّا نَحْنُ الْغَالِبِينَ } مُوسَى .
