فَجُمِعَ السَّحَرَةُ لِمِيقَاتِ يَوْمٍ مَعْلُومٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَجُمِعَ السَّحَرَة لِمِيقَاتِ يَوْم مَعْلُوم } يَقُول تَعَالَى ذِكْره : فَجُمِعَ الْحَاشِرُونَ الَّذِينَ بَعَثَهُمْ فِرْعَوْن بِحَشْرِ السَّحَرَة { لِمِيقَاتِ يَوْم مَعْلُوم } يَقُول : لِوَقْتٍ وَاعَدَ فِرْعَوْن لِمُوسَى الِاجْتِمَاع مَعَهُ فِيهِ مِنْ يَوْم مَعْلُوم , وَذَلِكَ يَوْم الزِّينَة { وَأَنْ يُحْشَر النَّاس ضُحًى } . 20 59
