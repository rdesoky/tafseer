وَيَضِيقُ صَدْرِي

{ وَيَضِيق صَدْرِي } مِنْ تَكْذِيبهمْ إِيَّايَ إِنْ كَذَّبُونِي . وَرُفِعَ قَوْله : { وَيَضِيق صَدْرِي } عَطْفًا بِهِ عَلَى أَخَاف , وَبِالرَّفْعِ فِيهِ قَرَأَتْهُ عَامَّة قُرَّاء الْأَمْصَار , وَمَعْنَاهُ : وَإِنِّي يَضِيق صَدْرِي .

وَلَا يَنْطَلِقُ لِسَانِي

وَقَوْله : { وَلَا يَنْطَلِق لِسَانِي } يَقُول : وَلَا يَنْطَلِق بِالْعِبَارَةِ عَمَّا تُرْسِلنِي بِهِ إِلَيْهِمْ , لِلْعِلَّةِ الَّتِي كَانَتْ بِلِسَانِهِ . وَقَوْله : { وَلَا يَنْطَلِق لِسَانِي } كَلَام مَعْطُوف بِهِ عَلَى يَضِيق.

فَأَرْسِلْ إِلَى هَارُونَ

وَقَوْله : { فَأَرْسِلْ إِلَى هَارُون } يَعْنِي هَارُون أَخَاهُ , وَلَمْ يَقُلْ : فَأَرْسِلْ إِلَيَّ هَارُون لِيُؤَازِرنِي وَلِيُعِينَنِي , إِذْ كَانَ مَفْهُومًا مَعْنَى الْكَلَام , وَذَلِكَ كَقَوْلِ الْقَائِل : لَوْ نَزَلَتْ بِنَا نَازِلَة لَفَزِعْنَا إِلَيْك , بِمَعْنَى : لَفَزِعْنَا إِلَيْك لِتُعِينَنَا .
