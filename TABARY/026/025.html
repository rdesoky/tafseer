قَالَ لِمَنْ حَوْلَهُ أَلَا تَسْتَمِعُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ لِمَنْ حَوْله أَلَا تَسْتَمِعُونَ } يَعْنِي تَعَالَى ذِكْره بِقَوْلِهِ { قَالَ لِمَنْ حَوْله أَلَا تَسْتَمِعُونَ } قَالَ فِرْعَوْن لِمَنْ حَوْله مِنْ قَوْمه : أَلَا تَسْتَمِعُونَ لِمَا يَقُول مُوسَى , فَأَخْبَرَ مُوسَى عَلَيْهِ السَّلَام الْقَوْم بِالْجَوَابِ عَنْ مَسْأَلَة فِرْعَوْن إِيَّاهُ وَقِيله لَهُ { وَمَا رَبّ الْعَالَمِينَ } لِيَفْهَم بِذَلِكَ قَوْم فِرْعَوْن مَقَالَته لِفِرْعَوْن , وَجَوَابه إِيَّاهُ عَمَّا سَأَلَهُ , إِذْ قَالَ لَهُمْ فِرْعَوْن { أَلَا تَسْتَمِعُونَ } إِلَى قَوْل مُوسَى ,
