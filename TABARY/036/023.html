أَأَتَّخِذُ مِنْ دُونِهِ آلِهَةً إِنْ يُرِدْنِ الرَّحْمَنُ بِضُرٍّ لَا تُغْنِ عَنِّي شَفَاعَتُهُمْ شَيْئًا وَلَا يُنْقِذُونِ

وَقَوْله : { أَأَتَّخِذُ مِنْ دُونه آلِهَة } يَقُول : أَأَعْبُدُ مِنْ دُون اللَّه آلِهَة , يَعْنِي مَعْبُودًا سِوَاهُ { إِنْ يُرِدْنِ الرَّحْمَن بِضُرٍّ } يَقُول : إِذْ مَسَّنِي الرَّحْمَن بِضُرٍّ وَشِدَّة { لَا تُغْنِ عَنِّي شَفَاعَتهمْ شَيْئًا } يَقُول : لَا تُغْنِي عَنِّي شَيْئًا بِكَوْنِهَا إِلَيَّ شُفَعَاء , وَلَا تَقْدِر عَلَى دَفْع ذَلِكَ الضُّرّ عَنِّي { وَلَا يُنْقِذُونَ } يَقُول : وَلَا يُخَلِّصُونِي مِنْ ذَلِكَ الضُّرّ إِذَا مَسَّنِي .
