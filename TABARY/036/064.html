اصْلَوْهَا الْيَوْمَ بِمَا كُنْتُمْ تَكْفُرُونَ

وَقَوْله : { اِصْلَوْهَا الْيَوْم بِمَا كُنْتُمْ تَكْفُرُونَ } يَقُول : اِحْتَرِقُوا بِهَا الْيَوْم وَرِدُوهَا ; يَعْنِي بِالْيَوْمِ : يَوْم الْقِيَامَة { بِمَا كُنْتُمْ تَكْفُرُونَ } : يَقُول : بِمَا كُنْتُمْ تَجْحَدُونَهَا فِي الدُّنْيَا , وَتَكْذِبُونَ بِهَا .
