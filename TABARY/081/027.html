إِنْ هُوَ إِلَّا ذِكْرٌ لِلْعَالَمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنْ هُوَ إِلَّا ذِكْر لِلْعَالَمِينَ } يَقُول تَعَالَى ذِكْره : { إِنْ } هَذَا الْقُرْآن , وَقَوْله : { هُوَ } مِنْ ذِكْر الْقُرْآن { إِلَّا ذِكْر لِلْعَالَمِينَ } يَقُول : إِلَّا تَذْكِرَة وَعِظَة لِلْعَالَمِينَ مِنْ الْجِنّ وَالْإِنْس
