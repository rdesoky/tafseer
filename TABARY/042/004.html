لَهُ مَا فِي السَّمَاوَاتِ وَمَا فِي الْأَرْضِ وَهُوَ الْعَلِيُّ الْعَظِيمُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { لَهُ مَا فِي السَّمَوَات وَمَا فِي الْأَرْض وَهُوَ الْعَلِيّ الْعَظِيم } يَقُول تَعَالَى ذِكْره : لِلَّهِ مُلْك { مَا فِي السَّمَوَات وَمَا فِي الْأَرْض } مِنَ الْأَشْيَاء كُلّهَا { وَهُوَ الْعَلِيّ } يَقُول : وَهُوَ ذُو عُلُوّ وَارْتِفَاع عَلَى كُلّ شَيْء , وَالْأَشْيَاء كُلّهَا دُونه ; لِأَنَّهُمْ فِي سُلْطَانه , جَارِيَة عَلَيْهِمْ قُدْرَته , مَاضِيَة فِيهِمْ مَشِيئَته { الْعَظِيم } الَّذِي لَهُ الْعَظَمَة وَالْكِبْرِيَاء وَالْجَبْرِيَّة .
