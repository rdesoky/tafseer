قُلْ لَنْ يُصِيبَنَا إِلَّا مَا كَتَبَ اللَّهُ لَنَا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قُلْ لَنْ يُصِيبنَا إِلَّا مَا كَتَبَ اللَّه لَنَا } يَقُول تَعَالَى ذِكْره مُؤَدِّبًا نَبِيّه مُحَمَّدًا صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : قُلْ يَا مُحَمَّد لِهَؤُلَاءِ الْمُنَافِقِينَ الَّذِينَ تَخَلَّفُوا عَنْك : { لَنْ يُصِيبنَا } أَيّهَا الْمُرْتَابُونَ فِي دِينهمْ { إِلَّا مَا كَتَبَ اللَّه لَنَا } فِي اللَّوْح الْمَحْفُوظ وَقَضَاهُ عَلَيْنَا .

هُوَ مَوْلَانَا

{ هُوَ مَوْلَانَا } يَقُول : هُوَ نَاصِرنَا عَلَى أَعْدَائِهِ .

وَعَلَى اللَّهِ فَلْيَتَوَكَّلِ الْمُؤْمِنُونَ

{ وَعَلَى اللَّه فَلْيَتَوَكَّلْ الْمُؤْمِنُونَ } يَقُول : وَعَلَى اللَّه فَلْيَتَوَكَّلْ الْمُؤْمِنُونَ , فَإِنَّهُمْ إِنْ يَتَوَكَّلُوا عَلَيْهِ وَلَمْ يَرْجُوا النَّصْر مِنْ عِنْد غَيْره وَلَمْ يَخَافُوا شَيْئًا غَيْره , يَكْفِهِمْ أُمُورهمْ وَيَنْصُرهُمْ عَلَى مَنْ بَغَاهُمْ وَكَادَهُمْ .
