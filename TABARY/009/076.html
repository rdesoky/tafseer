فَلَمَّا آتَاهُمْ مِنْ فَضْلِهِ بَخِلُوا بِهِ وَتَوَلَّوْا وَهُمْ مُعْرِضُونَ

{ فَلَمَّا آتَاهُمْ } اللَّه . { مِنْ فَضْله بَخِلُوا بِهِ } بِفَضْلِ اللَّه الَّذِي آتَاهُمْ , فَلَمْ يُصَدِّقُوا مِنْهُ وَلَمْ يَصِلُوا مِنْهُ قَرَابَة وَلَمْ يُنْفِقُوا مِنْهُ فِي حَقّ اللَّه. { وَتَوَلَّوْا } يَقُول : وَأَدْبَرُوا عَنْ عَهْدهمْ الَّذِي عَاهَدُوهُ اللَّه . { وَهُمْ مُعْرِضُونَ } عَنْهُ .
