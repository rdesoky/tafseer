خَالِدِينَ فِيهَا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { خَالِدِينَ فِيهَا } يَقُول تَعَالَى ذِكْره : { خَالِدِينَ فِيهَا } مَاكِثِينَ فِيهَا , يَعْنِي فِي الْجَنَّات.

أَبَدًا

{ أَبَدًا } لَا نِهَايَة لِذَلِكَ وَلَا حَدّ .

إِنَّ اللَّهَ عِنْدَهُ أَجْرٌ عَظِيمٌ

{ إِنَّ اللَّه عِنْده أَجْر عَظِيم } يَقُول : إِنَّ اللَّه عِنْده لِهَؤُلَاءِ الْمُؤْمِنِينَ الَّذِينَ نَعَتَهُمْ جَلَّ ثَنَاؤُهُ النَّعْت الَّذِي ذَكَرَ فِي هَذِهِ الْآيَة أَجْر : ثَوَاب عَلَى طَاعَتهمْ لِرَبِّهِمْ وَأَدَائِهِمْ مَا كَلَّفَهُمْ مِنْ الْأَعْمَال عَظِيم , وَذَلِكَ النَّعِيم الَّذِي وَعَدَهُمْ أَنْ يُعْطِيهِمْ فِي الْآخِرَة .
