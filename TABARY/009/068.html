وَعَدَ اللَّهُ الْمُنَافِقِينَ وَالْمُنَافِقَاتِ وَالْكُفَّارَ نَارَ جَهَنَّمَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَعَدَ اللَّه الْمُنَافِقِينَ وَالْمُنَافِقَات وَالْكُفَّار نَار جَهَنَّم } يَقُول تَعَالَى ذِكْره : { وَعَدَ اللَّه الْمُنَافِقِينَ وَالْمُنَافِقَات وَالْكُفَّار } بِاَللَّهِ { نَار جَهَنَّم } أَنْ يُصَلِّيهُمُوهَا جَمِيعًا .

خَالِدِينَ فِيهَا

{ خَالِدِينَ فِيهَا } يَقُول : مَاكِثِينَ فِيهَا أَبَدًا , لَا يَحْيَوْنَ فِيهَا وَلَا يَمُوتُونَ .

هِيَ حَسْبُهُمْ

{ هِيَ حَسْبهمْ } يَقُول : هِيَ كَافِيَتهمْ عِقَابًا وَثَوَابًا عَلَى كُفْرهمْ بِاَللَّهِ .

وَلَعَنَهُمُ اللَّهُ

{ وَلَعَنَهُمْ اللَّه } يَقُول : وَأَبْعَدَهُمْ اللَّه وَأَسْحَقهُمْ مِنْ رَحْمَته .

وَلَهُمْ عَذَابٌ مُقِيمٌ

{ وَلَهُمْ عَذَاب مُقِيم } يَقُول : وَلِلْفَرِيقَيْنِ جَمِيعًا , يَعْنِي مِنْ أَهْل النِّفَاق وَالْكُفْر عِنْد اللَّه , عَذَاب مُقِيم دَائِم , لَا يَزُول وَلَا يَبِيد .
