أَفَمَنْ كَانَ عَلَى بَيِّنَةٍ مِنْ رَبِّهِ كَمَنْ زُيِّنَ لَهُ سُوءُ عَمَلِهِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { أَفَمَنْ كَانَ عَلَى بَيِّنَة مِنْ رَبّه كَمَنْ زُيِّنَ لَهُ سُوء عَمَله } يَقُول تَعَالَى ذِكْره : { أَفَمَنْ كَانَ } عَلَى بُرْهَان وَحُجَّة وَبَيَان { مِنْ } أَمْر { رَبّه } وَالْعِلْم بِوَحْدَانِيَّتِهِ , فَهُوَ يَعْبُدهُ عَلَى بَصِيرَة مِنْهُ , بِأَنَّ لَهُ رَبًّا يُجَازِيه عَلَى طَاعَته إِيَّاهُ الْجَنَّة , وَعَلَى إِسَاءَته وَمَعْصِيَته إِيَّاهُ النَّار , { كَمَنْ زُيِّنَ لَهُ سُوء عَمَله } يَقُول : كَمَنْ حَسَّنَ لَهُ الشَّيْطَان قَبِيح عَمَله وَسَيِّئَهُ , فَأَرَاهُ جَمِيلًا , فَهُوَ عَلَى الْعَمَل بِهِ مُقِيم , وَقِيلَ : إِنَّ الَّذِي عُنِيَ بِقَوْلِهِ : { أَفَمَنْ كَانَ عَلَى بَيِّنَة مِنْ رَبّه } نَبِيّنَا عَلَيْهِ الصَّلَاة وَالسَّلَام , وَإِنَّ الَّذِي عُنِيَ بِقَوْلِهِ : { كَمَنْ زُيِّنَ لَهُ سُوء عَمَله } هُمْ الْمُشْرِكُونَ .

وَاتَّبَعُوا أَهْوَاءَهُمْ

يَقُول : وَاتَّبَعُوا مَا دَعَتْهُمْ إِلَيْهِ أَنْفُسهمْ مِنْ مَعْصِيَة اللَّه , وَعِبَادَة الْأَوْثَان مِنْ غَيْر أَنْ يَكُون عِنْدهمْ بِمَا يَعْمَلُونَ مِنْ ذَلِكَ بُرْهَان وَحُجَّة .
