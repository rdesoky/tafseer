أَلَا يَعْلَمُ مَنْ خَلَقَ وَهُوَ اللَّطِيفُ الْخَبِيرُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { أَلَا يَعْلَم مَنْ خَلَقَ وَهُوَ اللَّطِيف الْخَبِير } يَقُول تَعَالَى ذِكْره : { أَلَا يَعْلَم } الرَّبّ جَلَّ ثَنَاؤُهُ { مَنْ خَلَقَ } مَنْ خَلَقَهُ ؟ يَقُول : كَيْفَ يَخْفَى عَلَيْهِ خَلْقه الَّذِي خَلَقَ { وَهُوَ اللَّطِيف } بِعِبَادِهِ { الْخَبِير } بِهِمْ وَبِأَعْمَالِهِمْ .
