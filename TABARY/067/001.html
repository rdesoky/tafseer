تَبَارَكَ الَّذِي بِيَدِهِ الْمُلْكُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { تَبَارَكَ الَّذِي بِيَدِهِ الْمُلْك } يَعْنِي بِقَوْلِهِ تَعَالَى ذِكْره : { تَبَارَكَ } : تَعَاظَمَ وَتَعَالَى { الَّذِي بِيَدِهِ الْمُلْك } بِيَدِهِ مُلْك الدُّنْيَا وَالْآخِرَة وَسُلْطَانهمَا نَافِذ فِيهِمَا أَمْره وَقَضَاؤُهُ.

وَهُوَ عَلَى كُلِّ شَيْءٍ قَدِيرٌ

يَقُول : وَهُوَ عَلَى مَا يَشَاء فِعْله ذُو قُدْرَة لَا يَمْنَعهُ مِنْ فِعْله مَانِع , وَلَا يَحُول بَيْنه وَبَيْنه عَجْز .
