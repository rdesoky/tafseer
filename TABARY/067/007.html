إِذَا أُلْقُوا فِيهَا سَمِعُوا لَهَا شَهِيقًا

وَقَوْله : { إِذَا أُلْقُوا فِيهَا } يَعْنِي إِذَا أُلْقِيَ الْكَافِرُونَ فِي جَهَنَّم { سَمِعُوا لَهَا } يَعْنِي لِجَهَنَّم { شَهِيقًا } يَعْنِي بِالشَّهِيقِ : الصَّوْت الَّذِي يَخْرُج مِنَ الْجَوْف بِشِدَّةٍ كَصَوْتِ الْحِمَار , كَمَا قَالَ رُؤْبَة فِي صِفَة حِمَار : حَشْرَجَ فِي الْجَوْف سَحِيلًا أَوْ شَهَقْ حَتَّى يُقَال نَاهِق وَمَا نَهَقْ

وَهِيَ تَفُورُ

وَقَوْله : { وَهِيَ تَفُور } يَقُول : تَغْلِي . وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل. ذِكْر مَنْ قَالَ ذَلِكَ : 26732 - حَدَّثَنَا ابْن حُمَيْد , قَالَ : ثنا مِهْرَان , عَنْ سُفْيَان , عَنْ مُجَاهِد { سَمِعُوا لَهَا شَهِيقًا وَهِيَ تَفُور } يَقُول : تَغْلِي كَمَا يَغْلِي الْقَدْر .
