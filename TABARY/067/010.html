وَقَالُوا لَوْ كُنَّا نَسْمَعُ أَوْ نَعْقِلُ مَا كُنَّا فِي أَصْحَابِ السَّعِيرِ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَقَالُوا لَوْ كُنَّا نَسْمَع أَوْ نَعْقِل مَا كُنَّا مِنْ أَصْحَاب السَّعِير } يَقُول تَعَالَى ذِكْره : وَقَالَ الْفَوْج الَّذِي أُلْقِيَ فِي النَّار لِلْخَزَنَةِ : { لَوْ كُنَّا } فِي الدُّنْيَا { نَسْمَع أَوْ نَعْقِل } مِنْ النُّذُر مَا جَاءُونَا بِهِ مِنَ النَّصِيحَة , أَوْ نَعْقِل عَنْهُمْ مَا كَانُوا يَدْعُونَنَا إِلَيْهِ { مَا كُنَّا } الْيَوْم { فِي أَصْحَاب السَّعِير } يَعْنِي أَهْل النَّار .
