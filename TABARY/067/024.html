قُلْ هُوَ الَّذِي ذَرَأَكُمْ فِي الْأَرْضِ وَإِلَيْهِ تُحْشَرُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قُلْ هُوَ الَّذِي ذَرَأَكُمْ فِي الْأَرْض وَإِلَيْهِ تُحْشَرُونَ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد : قُلْ يَا مُحَمَّد , اللَّه { الَّذِي ذَرَأَكُمْ فِي الْأَرْض } يَقُول : اللَّه الَّذِي خَلَقَكُمْ فِي الْأَرْض { وَإِلَيْهِ تُحْشَرُونَ } يَقُول : وَإِلَى اللَّه تُحْشَرُونَ , فَتُجْمَعُونَ مِنْ قُبُوركُمْ لِمَوْقِفِ الْحِسَاب.
