قَالُوا بَلَى قَدْ جَاءَنَا نَذِيرٌ فَكَذَّبْنَا وَقُلْنَا مَا نَزَّلَ اللَّهُ مِنْ شَيْءٍ إِنْ أَنْتُمْ إِلَّا فِي ضَلَالٍ كَبِيرٍ

فَأَجَابَهُمُ الْمَسَاكِين { فَقَالُوا بَلَى قَدْ جَاءَنَا نَذِير } يُنْذِرنَا هَذَا , { فَكَذَّبْنَا } { وَقُلْنَا } لَهُ : { مَا نَزَّلَ اللَّه مِنْ شَيْء إِنْ أَنْتُمْ إِلَّا فِي ضَلَال كَبِير } يَقُول : فِي ذَهَاب عَنِ الْحَقّ بَعِيد .
