أَأَمِنْتُمْ مَنْ فِي السَّمَاءِ أَنْ يَخْسِفَ بِكُمُ الْأَرْضَ فَإِذَا هِيَ تَمُورُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { أَأَمِنْتُمْ مَنْ فِي السَّمَاء أَنْ يَخْسِف بِكُمُ الْأَرْض فَإِذَا هِيَ تَمُور } يَقُول تَعَالَى ذِكْره : { أَأَمِنْتُمْ مَنْ فِي السَّمَاء } أَيّهَا الْكَافِرُونَ { أَنْ يَخْسِف بِكُمُ الْأَرْض فَإِذَا هِيَ تَمُور } يَقُول : فَإِذَا الْأَرْض تَذْهَب بِكُمْ وَتَجِيء وَتَضْطَرِب .
