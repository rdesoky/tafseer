قُلْ هُوَ الَّذِي أَنْشَأَكُمْ وَجَعَلَ لَكُمُ السَّمْعَ وَالْأَبْصَارَ وَالْأَفْئِدَةَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قُلْ هُوَ الَّذِي أَنْشَأَكُمْ وَجَعَلَ لَكُمُ السَّمْع وَالْأَبْصَار وَالْأَفْئِدَة } يَقُول تَعَالَى ذِكْره : قُلْ يَا مُحَمَّد لِلَّذِينَ يُكَذِّبُونَ بِالْبَعْثِ مِنْ الْمُشْرِكِينَ . اللَّه الَّذِي أَنْشَأَكُمْ فَخَلَقَكُمْ , { وَجَعَلَ لَكُمُ السَّمْع } تَسْمَعُونَ بِهِ { وَالْأَبْصَار } تُبْصِرُونَ بِهَا { وَالْأَفْئِدَة } تَعْقِلُونَ بِهَا .

قَلِيلًا مَا تَشْكُرُونَ

يَقُول : قَلِيلًا مَا تَشْكُرُونَ رَبّكُمْ عَلَى هَذِهِ النِّعَم الَّتِي أَنْعَمَهَا عَلَيْكُمْ .
