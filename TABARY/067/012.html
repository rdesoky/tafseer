إِنَّ الَّذِينَ يَخْشَوْنَ رَبَّهُمْ بِالْغَيْبِ لَهُمْ مَغْفِرَةٌ وَأَجْرٌ كَبِيرٌ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِنَّ الَّذِينَ يَخْشَوْنَ رَبّهمْ بِالْغَيْبِ لَهُمْ مَغْفِرَة وَأَجْر كَبِير } يَقُول تَعَالَى ذِكْره : إِنَّ الَّذِينَ يَخَافُونَ رَبّهمْ بِالْغَيْبِ : يَقُول : وَهُمْ لَمْ يَرَوْهُ { لَهُمْ مَغْفِرَة } يَقُول : لَهُمْ عَفْو مِنَ اللَّه عَنْ ذُنُوبهمْ { وَأَجْر كَبِير } يَقُول : وَثَوَاب مِنَ اللَّه لَهُمْ عَلَى خَشْيَتهمْ إِيَّاهُ بِالْغَيْبِ جَزِيل .
