وَلِلَّذِينَ كَفَرُوا بِرَبِّهِمْ عَذَابُ جَهَنَّمَ وَبِئْسَ الْمَصِيرُ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلِلَّذِينَ كَفَرُوا بِرَبِّهِمْ عَذَاب جَهَنَّم وَبِئْسَ الْمَصِير } يَقُول تَعَالَى ذِكْره : { وَلِلَّذِينَ كَفَرُوا بِرَبِّهِمْ } الَّذِي خَلَقَهُمْ فِي الدُّنْيَا { عَذَاب جَهَنَّم } فِي الْآخِرَة { وَبِئْسَ الْمَصِير } يَقُول : وَبِئْسَ الْمَصِير عَذَاب جَهَنَّم.
