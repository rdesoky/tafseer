تَعْرِفُ فِي وُجُوهِهِمْ نَضْرَةَ النَّعِيمِ

وَقَوْله : { تَعْرِف فِي وُجُوههمْ نَضْرَة النَّعِيم } يَقُول تَعَالَى ذِكْره : تَعْرِف فِي الْأَبْرَار الَّذِينَ وَصَفَ اللَّه صِفَتهمْ نَضْرَة النَّعِيم , يَعْنِي حُسْنه وَبِرِيقِهِ وَتَلَأْلُؤُهُ . وَاخْتَلَفَتْ الْقُرَّاء فِي قِرَاءَة قَوْله : { تَعْرِف } فَقَرَأَتْهُ عَامَّة قُرَّاء الْأَمْصَار سِوَى أَبِي جَعْفَر الْقَارِئ { تَعْرِف فِي وُجُوههمْ } بِفَتْحِ التَّاء مِنْ تَعْرِف عَلَى وَجْه الْخِطَاب { نَضْرَة النَّعِيم } بِنَصْبِ نَضْرَة . وَقَرَأَ ذَلِكَ أَبُو جَعْفَر : " تُعْرَف " بِضَمِّ التَّاء عَلَى وَجْه مَا لَمْ يُسَمَّ فَاعِله , فِي وُجُوههمْ نَضْرَة النَّعِيم , بِرَفْعِ نَضْرَة . وَالصَّوَاب مِنْ الْقِرَاءَة فِي ذَلِكَ عِنْدنَا : مَا عَلَيْهِ قُرَّاء الْأَمْصَار , وَذَلِكَ فَتْح التَّاء مِنْ { تَعْرِف } وَنَصْب { نَضْرَة } .
