فَالْيَوْمَ الَّذِينَ آمَنُوا مِنَ الْكُفَّارِ يَضْحَكُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَالْيَوْم الَّذِينَ آمَنُوا مِنْ الْكُفَّار يَضْحَكُونَ } يَقُول تَعَالَى ذِكْره : { فَالْيَوْم } وَذَلِكَ يَوْم الْقِيَامَة { الَّذِينَ آمَنُوا } بِاَللَّهِ فِي الدُّنْيَا { مِنْ الْكُفَّار } فِيهَا { يَضْحَكُونَ }
