وَمَا يُكَذِّبُ بِهِ إِلَّا كُلُّ مُعْتَدٍ أَثِيمٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَمَا يُكَذِّب بِهِ إِلَّا كُلّ مُعْتَدٍ أَثِيم } يَقُول تَعَالَى ذِكْره : وَمَا يُكَذِّب بِيَوْمِ الدِّين { إِلَّا كُلّ مُعْتَدٍ } اِعْتَدَى عَلَى اللَّه فِي قَوْله , فَخَالَفَ أَمْره , { أَثِيم } بِرَبِّهِ , كَمَا : 28379 - حَدَّثَنَا بِشْر , قَالَ : ثَنَا يَزِيد , قَالَ : ثَنَا سَعِيد , عَنْ قَتَادَة { وَيْل يَوْمئِذٍ لِلْمُكَذِّبِينَ } قَالَ اللَّه : { وَمَا يُكَذِّب بِهِ إِلَّا كُلّ مُعْتَدٍ أَثِيم } أَيْ بِيَوْمِ الدِّين , إِلَّا كُلّ مُعْتَدٍ فِي قَوْله , أَثِيم بِرَبِّهِ .
