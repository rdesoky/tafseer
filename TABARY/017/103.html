فَأَرَادَ أَنْ يَسْتَفِزَّهُمْ مِنَ الْأَرْضِ فَأَغْرَقْنَاهُ وَمَنْ مَعَهُ جَمِيعًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَأَرَادَ أَنْ يَسْتَفِزّهُمْ مِنْ الْأَرْض فَأَغْرَقْنَاهُ وَمَنْ مَعَهُ جَمِيعًا } يَقُول تَعَالَى ذِكْره : فَأَرَادَ فِرْعَوْن أَنْ يَسْتَفِزّ مُوسَى وَبَنِي إِسْرَائِيل مِنْ الْأَرْض , { فَأَغْرَقْنَاهُ } فِي الْبَحْر , { وَمَنْ مَعَهُ } مِنْ جُنْده { جَمِيعًا } .
