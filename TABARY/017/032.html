وَلَا تَقْرَبُوا الزِّنَا إِنَّهُ كَانَ فَاحِشَةً وَسَاءَ سَبِيلًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى { وَلَا تَقْرَبُوا الزِّنَى إِنَّهُ كَانَ فَاحِشَة وَسَاءَ سَبِيلًا } . يَقُول تَعَالَى ذِكْره : وَقَضَى أَيْضًا أَنْ { لَا تَقْرَبُوا } أَيّهَا النَّاس { الزِّنَا إِنَّهُ كَانَ فَاحِشَة } يَقُول : إِنَّ الزِّنَا كَانَ فَاحِشَة { وَسَاءَ سَبِيلًا } يَقُول : وَسَاءَ طَرِيق الزِّنَا طَرِيقًا , لِأَنَّ طَرِيق أَهْل مَعْصِيَة اللَّه , وَالْمُخَالِفِينَ أَمْره , فَأَسْوِئْ بِهِ طَرِيقًا يُورِد صَاحِبه نَار جَهَنَّم .
