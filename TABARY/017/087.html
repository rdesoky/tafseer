إِلَّا رَحْمَةً مِنْ رَبِّكَ إِنَّ فَضْلَهُ كَانَ عَلَيْكَ كَبِيرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { إِلَّا رَحْمَة مِنْ رَبّك إِنَّ فَضْله كَانَ عَلَيْك كَبِيرًا } يَقُول عَزَّ وَجَلَّ : { وَلَئِنْ شِئْنَا لَنَذْهَبَنَّ } يَا مُحَمَّد { بِاَلَّذِي أَوْحَيْنَا إِلَيْك } وَلَكِنَّهُ لَا يَشَاء ذَلِكَ , رَحْمَة مِنْ رَبّك وَتَفَضُّلًا مِنْهُ عَلَيْك { إِنَّ فَضْله كَانَ عَلَيْك كَبِيرًا } بِاصْطِفَائِهِ إِيَّاكَ لِرِسَالَتِهِ , وَإِنْزَاله عَلَيْك كِتَابه , وَسَائِر نِعَمه عَلَيْك الَّتِي لَا تُحْصَى .
