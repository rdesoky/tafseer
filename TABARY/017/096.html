قُلْ كَفَى بِاللَّهِ شَهِيدًا بَيْنِي وَبَيْنَكُمْ إِنَّهُ كَانَ بِعِبَادِهِ خَبِيرًا بَصِيرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى { قُلْ كَفَى بِاَللَّهِ شَهِيدًا بَيْنِي وَبَيْنكُمْ إِنَّهُ كَانَ بِعِبَادِهِ خَبِيرًا بَصِيرًا } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ : قُلْ يَا مُحَمَّد لِلْقَائِلِينَ لَك : { أَبَعَثَ اللَّه بَشَرًا رَسُولًا } { كَفَى بِاَللَّهِ شَهِيدًا بَيْنِي وَبَيْنكُمْ } فَإِنَّهُ نِعْمَ الْكَافِي وَالْحَاكِم { إِنَّهُ كَانَ بِعِبَادِهِ خَبِيرًا } يَقُول : إِنَّ اللَّه بِعِبَادِهِ ذُو خِبْرَة وَعِلْم بِأُمُورِهِمْ وَأَفْعَالهمْ , وَالْمُحِقّ مِنْهُمْ وَالْمُبْطِل , وَالْمُهْدِي وَالضَّالّ { بَصِيرًا } بِتَدْبِيرِهِمْ وَسِيَاسَتهمْ وَتَصْرِيفهمْ فِيمَا شَاءَ , وَكَيْف شَاءَ وَأَحَبَّ , لَا يَخْفَى عَلَيْهِ شَيْء مِنْ أُمُورهمْ , وَهُوَ مَجَاز جَمِيعهمْ بِمَا قَدِمَ عِنْد وُرُودهمْ عَلَيْهِ .
