وَكَانَ يَأْمُرُ أَهْلَهُ بِالصَّلَاةِ وَالزَّكَاةِ وَكَانَ عِنْدَ رَبِّهِ مَرْضِيًّا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَكَانَ يَأْمُر أَهْله بِالصَّلَاةِ وَالزَّكَاة وَكَانَ عِنْد رَبّه مَرْضِيًّا } يَقُول تَعَالَى ذِكْره : { وَكَانَ يَأْمُر أَهْله ب } إِقَامَة { الصَّلَاة و } إِيتَاء { الزَّكَاة وَكَانَ عِنْد رَبّه مَرْضِيًّا } عَمَله , مَحْمُودًا فِيمَا كَلَّفَهُ رَبّه , غَيْر مُقَصِّر فِي طَاعَته .
