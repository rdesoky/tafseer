قَالَتْ أَنَّى يَكُونُ لِي غُلَامٌ وَلَمْ يَمْسَسْنِي بَشَرٌ وَلَمْ أَكُ بَغِيًّا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَتْ أَنَّى يَكُون لِي غُلَام وَلَمْ يَمْسَسْنِي بَشَر وَلَمْ أَكُ بَغِيًّا } يَقُول تَعَالَى ذِكْره : قَالَتْ مَرْيَم لِجِبْرِيل : { أَنَّى يَكُون لِي غُلَام } مِنْ أَيّ وَجْه يَكُون لِي غُلَام ؟ أَمِنْ قِبَل زَوْج أَتَزَوَّج , فَأُرْزَقهُ مِنْهُ , أَمْ يَبْتَدِئ اللَّه فِي خَلْقه اِبْتِدَاء { وَلَمْ يَمْسَسْنِي بَشَر } مِنْ وَلَد آدَم بِنِكَاحٍ حَلَال { وَلَمْ أَكُ } إِذْ لَمْ يَمْسَسْنِي مِنْهُمْ أَحَد عَلَى وَجْه الْحَلَال { بَغِيًّا } بَغَيْت فَفَعَلْت ذَلِكَ مِنْ الْوَجْه الْحَرَام , فَحَمَلَتْهُ مِنْ زِنًا , كَمَا : 17773 - حَدَّثَنَا مُوسَى , قَالَ : ثنا عَمْرو , قَالَ : ثنا أَسْبَاط , عَنْ السُّدِّيّ { وَلَمْ أَكُ بَغِيًّا } يَقُول : زَانِيَة .
