وَيَقُولُونَ مَتَى هَذَا الْوَعْدُ إِنْ كُنْتُمْ صَادِقِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَيَقُولُونَ مَتَى هَذَا الْوَعْد إِنْ كُنْتُمْ صَادِقِينَ } { قُلْ لَكُمْ مِيعَاد يَوْم لَا تَسْتَأْخِرُونَ عَنْهُ سَاعَة وَلَا تَسْتَقْدِمُونَ } يَقُول تَعَالَى ذِكْره : وَيَقُول هَؤُلَاءِ الْمُشْرِكُونَ بِاللَّهِ إِذَا سَمِعُوا وَعِيدَ اللَّه الْكُفَّار وَمَا هُوَ فَاعِل بِهِمْ فِي مُعَادهمْ مِمَّا أَنْزَلَ اللَّه فِي كِتَابه : { مَتَى هَذَا الْوَعْد } جَائِيًا , وَفِي أَيّ وَقْت هُوَ كَائِن { إِنْ كُنْتُمْ } فِيمَا تَعِدُونَنَا مِنْ ذَلِكَ { صَادِقِينَ } أَنَّهُ كَائِن ,
