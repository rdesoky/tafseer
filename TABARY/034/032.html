قَالَ الَّذِينَ اسْتَكْبَرُوا لِلَّذِينَ اسْتُضْعِفُوا أَنَحْنُ صَدَدْنَاكُمْ عَنِ الْهُدَى بَعْدَ إِذْ جَاءَكُمْ بَلْ كُنْتُمْ مُجْرِمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ الَّذِينَ اسْتَكْبَرُوا لِلَّذِينَ اسْتُضْعِفُوا أَنَحْنُ صَدَدْنَاكُمْ عَنِ الْهُدَى بَعْدَ إِذْ جَاءَكُمْ بَلْ كُنْتُمْ مُجْرِمِينَ } يَقُول تَعَالَى ذِكْره : { قَالَ الَّذِينَ اسْتَكْبَرُوا } فِي الدُّنْيَا , فَرَأَسُوا فِي الضَّلَالَة وَالْكُفْر بِاللَّهِ { لِلَّذِينَ اسْتُضْعِفُوا } فِيهَا فَكَانُوا أَتْبَاعًا لِأَهْلِ الضَّلَالَة مِنْهُمْ إِذْ قَالُوا لَهُمْ { لَوْلَا أَنْتُمْ لَكُنَّا مُؤْمِنِينَ أَنَحْنُ صَدَدْنَاكُمْ عَنِ الْهُدَى } وَمَنَعْنَاكُمْ مِنْ اتِّبَاع الْحَقّ { بَعْدَ إِذْ جَاءَكُمْ } مِنْ عِنْد اللَّه , يُبَيِّن لَكُمْ { بَلْ كُنْتُمْ مُجْرِمِينَ } فَمَنَعَكُمْ إِيثَاركُمْ الْكُفْرَ بِاللَّهِ عَلَى الْإِيمَان مِنْ اتِّبَاع الْهُدَى , وَالْإِيمَان بِاللَّهِ وَرَسُوله .
