وَقَالَ الَّذِينَ كَفَرُوا هَلْ نَدُلُّكُمْ عَلَى رَجُلٍ يُنَبِّئُكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّقٍ إِنَّكُمْ لَفِي خَلْقٍ جَدِيدٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَقَالَ الَّذِينَ كَفَرُوا هَلْ نَدُلّكُمْ عَلَى رَجُل يُنَبِّئُكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّق إِنَّكُمْ لَفِي خَلْق جَدِيد } يَقُول تَعَالَى ذِكْره : وَقَالَ الَّذِينَ كَفَرُوا بِاللَّهِ وَبِرَسُولِهِ مُحَمَّد , مُتَعَجِّبِينَ مِنْ وَعْده إِيَّاهُمُ الْبَعْث بَعْد الْمَمَات بَعْضهمْ لِبَعْضٍ : { هَلْ نَدُلّكُمْ } أَيّهَا النَّاس { عَلَى رَجُل يُنَبِّئكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّق إِنَّكُمْ لَفِي خَلْق جَدِيد } يَقُول : يُخْبِركُمْ أَنَّكُمْ بَعْد تَقَطُّعكِمْ فِي الْأَرْض بَلَاء وَبَعْد مَصِيركُمْ فِي التُّرَاب رُفَاتًا , عَائِدُونَ كَهَيْئَتِكُمْ قَبْل الْمَمَات خَلْقًا جَدِيدًا , كَمَا : 21920 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة { وَقَالَ الَّذِينَ كَفَرُوا هَلْ نَدُلّكُمْ عَلَى رَجُلٍ يُنَبِّئكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّق } قَالَ : ذَلِكَ مُشْرِكُو قُرَيْش وَالْمُشْرِكُونَ مِنَ النَّاس , { يُنَبِّئكُمْ إِذَا مُزِّقْتُمْ كُلَّ مُمَزَّق } : إِذَا أَكَلَتْكُمْ الْأَرْض , وَصِرْتُمْ رُفَاتًا وَعِظَامًا , وَقَطَّعَتْكُمْ السِّبَاع وَالطَّيْر { إِنَّكُمْ لَفِي خَلْق جَدِيد } سَتَحْيَوْنَ وَتُبْعَثُونَ . 21921 -حَدَّثَنِي يُونُس , قَالَ : أَخْبَرَنَا ابْن وَهْب , قَالَ : قَالَ ابْن زَيْد , فِي قَوْله : { هَلْ نَدُلّكُمْ عَلَى رَجُل } إِلَى { خَلْق جَدِيد } قَالَ : يَقُول : { إِذَا مُزِّقْتُمْ } : وَإِذَا بُلِيتُمْ وَكُنْتُمْ عِظَامًا وَتُرَابًا وَرُفَاتًا , ذَلِكَ { كُلّ مُمَزَّق إِنَّكُمْ لَفِي خَلْق جَدِيد } قَالَ : يُنَبِّئكُمْ إِنَّكُمْ , فَكَسَرَ إِنْ وَلَمْ يُعْمِل يُنَبِّئكُمْ فِيهَا , وَلَكِنْ ابْتَدَأَ بِهَا ابْتِدَاء ; لِأَنَّ النَّبَأ خَبَر وَقَوْل , فَالْكَسْر فِي إِنَّ لِمَعْنَى الْحِكَايَة فِي قَوْله : { يُنَبِّئكُمْ } دُون لَفْظه , كَأَنَّهُ قِيلَ : يَقُول لَكُمْ : { إِنَّكُمْ لَفِي خَلْق جَدِيد } .
