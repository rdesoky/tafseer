فَالْيَوْمَ لَا يَمْلِكُ بَعْضُكُمْ لِبَعْضٍ نَفْعًا وَلَا ضَرًّا وَنَقُولُ لِلَّذِينَ ظَلَمُوا ذُوقُوا عَذَابَ النَّارِ الَّتِي كُنْتُمْ بِهَا تُكَذِّبُونَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَالْيَوْم لَا يَمْلِك بَعْضكُمْ لِبَعْضٍ نَفْعًا وَلَا ضَرًّا وَنَقُول لِلَّذِينَ ظَلَمُوا ذُوقُوا عَذَابَ النَّار الَّتِي كُنْتُمْ بِهَا تُكَذِّبُونَ } يَقُول تَعَالَى ذِكْره : فَالْيَوْم لَا يَمْلِك بَعْضكُمْ أَيّهَا الْمَلَائِكَة لِلَّذِينَ كَانُوا فِي الدُّنْيَا يَعْبُدُونَكُمْ نَفْعًا يَنْفَعُونَكُمْ بِهِ وَلَا ضَرًّا يَنَالُونَكُمْ بِهِ , أَوْ تَنَالُونَهُمْ بِهِ { وَنَقُول لِلَّذِينَ ظَلَمُوا } يَقُول : وَنَقُول لِلَّذِينَ عَبَدُوا غَيْرَ اللَّه فَوَضَعُوا الْعِبَادَةَ فِي غَيْر مَوْضِعهَا , وَجَعَلُوهَا لِغَيْرِ مَنْ تَنْبَغِي أَنْ تَكُونَ لَهُ : { ذُوقُوا عَذَابَ النَّار الَّتِي كُنْتُمْ بِهَا } فِي الدُّنْيَا { تُكَذِّبُونَ } فَقَدْ وَرَدْتُمُوهَا .
