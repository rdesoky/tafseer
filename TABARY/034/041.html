قَالُوا سُبْحَانَكَ أَنْتَ وَلِيُّنَا مِنْ دُونِهِمْ بَلْ كَانُوا يَعْبُدُونَ الْجِنَّ أَكْثَرُهُمْ بِهِمْ مُؤْمِنُونَ

{ قَالُوا سُبْحَانَك } رَبّنَا , تَنْزِيهًا لَك وَتَبْرِئَةً مِمَّا أَضَافَ إِلَيْك هَؤُلَاءِ مِنْ الشُّرَكَاء وَالْأَنْدَاد { أَنْتَ وَلِيّنَا مِنْ دُونهمْ } لَا نَتَّخِذ وَلِيًّا دُونَك { بَلْ كَانُوا يَعْبُدُونَ الْجِنَّ } وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل . ذِكْر مَنْ قَالَ ذَلِكَ : 22065 - حَدَّثَنَا بِشْر , قَالَ : ثنا يَزِيد , قَالَ : ثنا سَعِيد , عَنْ قَتَادَة , قَوْله : { وَيَوْم نَحْشُرهُمْ جَمِيعًا ثُمَّ نَقُول لِلْمَلَائِكَةِ أَهَؤُلَاءِ إِيَّاكُمْ كَانُوا يَعْبُدُونَ } اسْتِفْهَام , كَقَوْلِهِ لِعِيسَى : { أَأَنْت قُلْت لِلنَّاسِ اتَّخِذُونِي وَأُمِّيَ إِلَهَيْنِ مِنْ دُون اللَّه } 5 116 وَقَوْله : { أَكْثَرهمْ بِهِمْ مُؤْمِنُونَ } يَقُول : أَكْثَرهمْ بِالْجِنِّ مُصَدِّقُونَ , يَزْعُمُونَ أَنَّهُمْ بَنَات اللَّه , تَعَالَى اللَّه عَمَّا يَقُولُونَ عُلُوًّا كَبِيرًا .
