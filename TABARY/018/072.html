قَالَ أَلَمْ أَقُلْ إِنَّكَ لَنْ تَسْتَطِيعَ مَعِيَ صَبْرًا

الْقَوْل فِي تَأْوِيل تَعَالَى : { قَالَ أَلَمْ أَقُلْ إِنَّك لَنْ تَسْتَطِيع مَعِيَ صَبْرًا } يَقُول عَزَّ ذِكْره : { قَالَ } الْعَالِم لِمُوسَى إِذْ قَالَ لَهُ مَا قَالَ { أَلَمْ أَقُلْ إِنَّك لَنْ تَسْتَطِيع مَعِيَ صَبْرًا } عَلَى مَا تَرَى مِنْ أَفْعَالِي , لِأَنَّك تَرَى مَا لَمْ تُحِطْ بِهِ خُبْرًا .
