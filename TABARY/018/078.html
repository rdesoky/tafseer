قَالَ هَذَا فِرَاقُ بَيْنِي وَبَيْنِكَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ هَذَا فِرَاق بَيْنِي وَبَيْنك } سَأُنَبِّئُك بِتَأْوِيلِ مَا لَمْ تَسْتَطِعْ عَلَيْهِ صَبْرًا } يَقُول تَعَالَى ذِكْره : قَالَ صَاحِب مُوسَى لِمُوسَى : هَذَا الَّذِي قُلْته وَهُوَ قَوْله { لَوْ شِئْت لَاِتَّخَذْت عَلَيْهِ أَجْرًا } { فِرَاق بَيْنِي وَبَيْنك } يَقُول : فِرْقَة مَا بَيْنِي وَبَيْنك : أَيْ مَفْرِق بَيْنِي وَبَيْنك.

سَأُنَبِّئُكَ بِتَأْوِيلِ مَا لَمْ تَسْتَطِعْ عَلَيْهِ

{ سَأُنَبِّئُك } يَقُول : سَأُخْبِرُك { بِتَأْوِيلِ مَا لَمْ تَسْتَطِيع عَلَيْهِ صَبْرًا } يَقُول : بِمَا يَئُول إِلَيْهِ عَاقِبَة أَفْعَالِي الَّتِي فَعَلْتهَا , فَلَمْ تَسْتَطِعْ عَلَيَّ تَرْك الْمَسْأَلَة عَنْهَا , وَعَنْ النَّكِير عَلَيَّ فِيهَا صَبْرًا , وَاَللَّه أَعْلَم.
