وَاضْرِبْ لَهُمْ مَثَلًا رَجُلَيْنِ جَعَلْنَا لِأَحَدِهِمَا جَنَّتَيْنِ مِنْ أَعْنَابٍ وَحَفَفْنَاهُمَا بِنَخْلٍ وَجَعَلْنَا بَيْنَهُمَا زَرْعًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَاضْرِبْ لَهُمْ مَثَلًا رَجُلَيْنِ جَعَلْنَا لِأَحَدِهِمَا جَنَّتَيْنِ مِنْ أَعْنَاب وَحَفَفْنَاهُمَا بِنَخْلٍ وَجَعَلْنَا بَيْنهمَا زَرْعًا } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : وَاضْرِبْ يَا مُحَمَّد لِهَؤُلَاءِ الْمُشْرِكِينَ بِاَللَّهِ , الَّذِينَ سَأَلُوك أَنْ تَطْرُد الَّذِينَ يَدْعُونَ رَبّهمْ بِالْغَدَاةِ وَالْعَشِيّ يُرِيدُونَ وَجْهه , { مَثَلًا } مَثَل { رَجُلَيْنِ جَعَلْنَا لِأَحَدِهِمَا جَنَّتَيْنِ } أَيْ جَعَلْنَا لَهُ بَسَاتِين مِنْ كُرُوم { وَحَفَفْنَاهُمَا بِنَخْلٍ } يَقُول : وَأَطَفْنَا هَذَيْنِ الْبَسَاتِين بِنَخْلٍ . وَقَوْله : { وَجَعَلْنَا بَيْنهمَا زَرْعًا } يَقُول : وَجَعَلْنَا وَسَط هَذَيْنِ الْبَسَاتِين زَرْعًا .
