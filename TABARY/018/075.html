قَالَ أَلَمْ أَقُلْ لَكَ إِنَّكَ لَنْ تَسْتَطِيعَ مَعِيَ صَبْرًا

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قَالَ أَلَمْ أَقُلْ لَك إِنَّك لَنْ تَسْتَطِيع مَعِيَ صَبْرًا } يَقُول تَعَالَى ذِكْره : قَالَ الْعَالِم لِمُوسَى { أَلَمْ أَقُلْ لَك إِنَّك لَنْ تَسْتَطِيع مَعِيَ صَبْرًا } عَلَى مَا تَرَى مِنْ أَفْعَالِي الَّتِي لَمْ تُحِطْ بِهَا خُبْرًا .
