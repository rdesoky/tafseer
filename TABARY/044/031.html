مِنْ فِرْعَوْنَ إِنَّهُ كَانَ عَالِيًا مِنَ الْمُسْرِفِينَ

وَقَوْله : { مِنْ فِرْعَوْن إِنَّهُ كَانَ عَالِيًا مِنَ الْمُسْرِفِينَ } يَقُول تَعَالَى ذِكْره : وَلَقَدْ نَجَّيْنَا بَنِي إِسْرَائِيل مِنَ الْعَذَاب مِنْ فِرْعَوْن , فَقَوْله : { مِنْ فِرْعَوْن } مُكَرَّرَة عَلَى قَوْله : { مِنَ الْعَذَاب الْمُهِين } مُبْدَلَة مِنَ الْأُولَى , وَيَعْنِي بِقَوْلِهِ : { إِنَّهُ كَانَ عَالِيًا مِنَ الْمُسْرِفِينَ } إِنَّهُ كَانَ جَبَّارًا مُسْتَعْلِيًا مُسْتَكْبِرًا عَلَى رَبّه , { مِنَ الْمُسْرِفِينَ } يَعْنِي : مِنْ الْمُتَجَاوِزِينَ مَا لَيْسَ لَهُمْ تَجَاوُزه . وَإِنَّمَا يَعْنِي جَلَّ ثَنَاؤُهُ أَنَّهُ كَانَ ذَا اعْتِدَاء فِي كُفْره , وَاسْتِكْبَار عَلَى رَبّه جَلَّ ثَنَاؤُهُ .
