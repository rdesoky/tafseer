قُلْ إِنَّ صَلَاتِي وَنُسُكِي وَمَحْيَايَ وَمَمَاتِي لِلَّهِ رَبِّ الْعَالَمِينَ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { قُلْ إِنَّ صَلَاتِي وَنُسُكِي وَمَحْيَايَ وَمَمَاتِي لِلَّهِ رَبّ الْعَالَمِينَ لَا شَرِيك لَهُ وَبِذَلِكَ أُمِرْت وَأَنَا أَوَّل الْمُسْلِمِينَ } يَقُول تَعَالَى ذِكْره لِنَبِيِّهِ مُحَمَّد صَلَّى اللَّه عَلَيْهِ وَسَلَّمَ : { قُلْ } يَا مُحَمَّد لِهَؤُلَاءِ الْعَادِلِينَ بِرَبِّهِمْ الْأَوْثَان وَالْأَصْنَام , الَّذِينَ يَسْأَلُونَك أَنْ تَتَّبِعَ أَهْوَاءَهُمْ عَلَى الْبَاطِل مِنْ عِبَادَة الْآلِهَة وَالْأَوْثَان : { إِنَّ صَلَاتِي وَنُسُكِي } يَقُول : وَذِبْحِي . { وَمَحْيَايَ } يَقُول : وَحَيَاتِي . { وَمَمَاتِي } يَقُول : وَوَفَاتِي . { لِلَّهِ رَبّ الْعَالَمِينَ } يَعْنِي أَنَّ ذَلِكَ كُلّه لَهُ خَالِصًا دُون مَا أَشْرَكْتُمْ بِهِ أَيّهَا الْمُشْرِكُونَ مِنْ الْأَوْثَان .
