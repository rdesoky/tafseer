وَقَالَ مُوسَى إِنْ تَكْفُرُوا أَنْتُمْ وَمَنْ فِي الْأَرْضِ جَمِيعًا فَإِنَّ اللَّهَ لَغَنِيٌّ حَمِيدٌ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَقَالَ مُوسَى إِنْ تَكْفُرُوا أَنْتُمْ وَمَنْ فِي الْأَرْض جَمِيعًا فَإِنَّ اللَّه لَغَنِيّ حَمِيد } يَقُول تَعَالَى ذِكْره : { وَقَالَ مُوسَى } لِقَوْمِهِ : { إِنْ تَكْفُرُوا } أَيّهَا الْقَوْم , فَتَجْحَدُوا نِعْمَة اللَّه الَّتِي أَنْعَمَهَا عَلَيْكُمْ أَنْتُمْ , وَيَفْعَل فِي ذَلِكَ مِثْل فِعْلكُمْ { مَنْ فِي الْأَرْض جَمِيعًا فَإِنَّ اللَّه لَغَنِيّ } عَنْكُمْ وَعَنْهُمْ مِنْ جَمِيع خَلْقه , لَا حَاجَة بِهِ إِلَى شُكْركُمْ إِيَّاهُ عَلَى نِعَمه عِنْد جَمِيعكُمْ , { حَمِيد } ذُو حَمْد إِلَى خَلْقه بِمَا أَنْعَمَ بِهِ عَلَيْهِمْ . كَمَا : 15609 - حَدَّثَنِي الْمُثَنَّى , قَالَ : ثَنَا إِسْحَاق , قَالَ : ثَنَا عَبْد اللَّه بْن هَاشِم , قَالَ : أَخْبَرَنَا سَيْف , عَنْ أَبِي رَوْق , عَنْ أَبِي أَيُّوب , عَنْ عَلِيّ : { فَإِنَّ اللَّه لَغَنِيّ } قَالَ : غَنِيّ عَنْ خَلْقه , { حَمِيد } قَالَ : مُسْتَحْمَد إِلَيْهِمْ .
