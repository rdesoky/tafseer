فَفَتَحْنَا أَبْوَابَ السَّمَاءِ بِمَاءٍ مُنْهَمِرٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { فَفَتَحْنَا أَبْوَاب السَّمَاء بِمَاءٍ مُنْهَمِر } يَقُول تَعَالَى ذِكْرُهُ : { فَفَتَحْنَا } لَمَّا دَعَانَا نُوح مُسْتَغِيثًا بِنَا عَلَى قَوْمه { أَبْوَابَ السَّمَاء بِمَاءٍ مُنْهَمِر } وَهُوَ الْمُنْدَفِق , كَمَا قَالَ امْرُؤُ الْقَيْس فِي صِفَة غَيْث : رَاحَ تَمْرِيهِ الصَّبَا ثُمَّ انْتَحَى فِيهِ شُؤْبُوبُ جَنُوبٍ مُنْهَمِرٍ وَبِنَحْوِ الَّذِي قُلْنَا فِي ذَلِكَ قَالَ أَهْل التَّأْوِيل. ذِكْرُ مَنْ قَالَ ذَلِكَ : 25333 - حَدَّثَنَا ابْن حُمَيْد , قَالَ : ثنا مِهْرَان , عَنْ سُفْيَان { بِمَاءٍ مُنْهَمِرٍ } قَالَ : يَنْصَبُّ انْصِبَابًا.
