وَلَقَدْ يَسَّرْنَا الْقُرْآنَ لِلذِّكْرِ فَهَلْ مِنْ مُدَّكِرٍ

الْقَوْل فِي تَأْوِيل قَوْله تَعَالَى : { وَلَقَدْ يَسَّرْنَا الْقُرْآن لِلذِّكْرِ فَهَلْ مِنْ مُدَّكِر } يَقُول تَعَالَى ذِكْرُهُ : وَلَقَدْ سَهَّلْنَا الْقُرْآنَ وَهَوَّنَّاهُ لِمَنْ أَرَادَ التَّذَكُّر بِهِ وَالِاتِّعَاظ { فَهَلْ مِنْ مُدَّكِر } يَقُول : فَهَلْ مِنْ مُتَّعِظ وَمُنْزَجِر بِآيَاتِهِ .
