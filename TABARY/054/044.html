أَمْ يَقُولُونَ نَحْنُ جَمِيعٌ مُنْتَصِرٌ

وَقَوْله : { أَمْ يَقُولُونَ نَحْنُ جَمِيع مُنْتَصِر } يَقُول تَعَالَى ذِكْرُهُ : أَيَقُولُ هَؤُلَاءِ الْكُفَّار مِنْ قُرَيْش : نَحْنُ جَمِيع مُنْتَصِر مِمَّنْ قَصَدَنَا بِسُوءٍ وَمَكْرُوه , وَأَرَادَ حَرْبَنَا وَتَفْرِيقَ جَمْعِنَا , فَقَالَ اللَّه جَلَّ ثَنَاؤُهُ : سَيُهْزَمُ الْجَمْع يَعْنِي جَمْعَ كُفَّار قُرَيْش { وَيُوَلُّونَ الدُّبُر }
